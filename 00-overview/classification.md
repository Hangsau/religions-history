# 宗教分類框架

> 多種分類視角並陳。分類是研究工具，不是真理。

## 分類視角

### 1. 按起源地

| 區域 | 宗教 |
|------|------|
| 兩河 / 西亞 | 蘇美、巴比倫、猶太、基督、伊斯蘭、瑣羅亞斯德、巴哈伊 |
| 北非 / 埃及 | 古埃及、科普特正教、伊斯蘭（北非分支）|
| 南亞 | 印度教、佛教、耆那、錫克 |
| 東亞 | 儒、道、佛（漢傳）、神道 |
| 非洲 | 傳統部落信仰、萬物有靈、巫毒 |
| 大洋洲 | 澳洲原住民、波里尼西亞 |
| 美洲 | 瑪雅、阿茲特克、印加、北美原住民自然信仰 |
| 現代 | 摩門、統一教、Scientology、新異教 |

### 2. 按一神 / 多神

| 類型 | 宗教 |
|------|------|
| 嚴格一神教 | 猶太、基督、伊斯蘭、錫克、巴哈伊 |
| 主流一神但有層級 | 部分印度教派（毗濕奴派 / 濕婆派）|
| 多神 | 古希臘、北歐、古代美索、古代埃及、神道、印度教（綜合）|
| 萬物有靈 / 泛靈 | 非洲、澳洲原住民、波里尼西亞、薩滿傳統 |
| 無神 / 自然主義 | 佛教（教義爭議，部分學派）、耆那、原始佛教 |
| 不可知 / 開放 | 部分新興宗教 |

### 3. 按組織型態

| 型態 | 範例 |
|------|------|
| 教會制 | 天主教、東正教 |
| 會眾制 | 新教各派 |
| 教團制 | 早期佛教、耆那、藏傳佛教 |
| 民族宗教 | 猶太（廣義）、神道、印度教 |
| 擴散信仰 | 民間信仰、無固定組織 |

### 4. 按救贖觀

| 路線 | 宗教 |
|------|------|
| 因信稱義 | 基督新教 |
| 因行稱義 | 天主教、東正教、伊斯蘭、猶太（部分）|
| 解脫輪迴 | 印度教、佛教、耆那 |
| 與神合一 | 蘇菲、密教、巴哈伊、克裡希那 |
| 民族救贖 | 部分猶太派、神道國學派 |
| 無救贖觀 | 儒（人文主義取向）、部分原始信仰 |

## 爭議與限制

- 「一神 / 多神」是西方神學框架，套用印度教 / 神道時容易失真
- 「宗教」定義本身爭議（儒教算宗教嗎？民間信仰算嗎？）
- 現代分類常帶殖民視角（基督新教學者早期分類標準偏基督教中心）
