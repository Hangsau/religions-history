## 2. 處理原則

每經典啟動翻譯前：

1. **判定類型**（A / B / C / D / E）
2. **選定工作文本**（最早 / 最完整 / 最流通）
3. **盤點多語版本**（蛛絲馬跡）
4. **寫版本檔**（`translations/[經典]/00-version.md`）
5. **AI 翻譯 + 交叉驗證**（按類型調整）

## 3. B 類經典特別處理

對於 B 類（原語言晚於翻譯）：

- **承認「原文」可能是漢文 / 中亞漢化佛教圈**
- 採用「最早完整翻譯」為工作文本
- 對照原語言版本（即使晚）
- 寫出「學界爭論」段，明確標明未定論處
