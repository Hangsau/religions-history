## 1. 史前 / 兩河 / 西亞

### 1.1 蘇美宗教
- **起源**：~3500 BC / 美索不達米亞南部（伊拉克）
- **核心信條**：多神，城市神守護（恩利爾、恩基、伊南娜）；廟塔（ziggurat）崇拜
- **核心經典**：楔形文字泥板（吉爾伽美什史詩、創世神話）
- **當代規模**：0（消亡，~2000 BC 被巴比倫吸收）
- **狀態**：消亡 / 神話遺產保留在後續宗教

### 1.2 古巴比倫宗教
- **起源**：~1894 BC / 巴比倫
- **核心信條**：馬杜克為主神；新年節（Akitu）；占星術發達
- **核心經典**：埃努瑪·埃利什（創世史詩）、漢摩拉比法典（世俗）
- **狀態**：消亡（~539 BC 波斯征服）

### 1.3 亞述宗教
- **起源**：~2500 BC / 亞述
- **核心信條**：阿舒爾為主神；軍事擴張伴宗教擴張
- **狀態**：消亡（~612 BC）

### 1.4 赫梯宗教
- **起源**：~1600 BC / 安納托利亞
- **核心信條**：自然神、storm god Teshub；契約文化
- **狀態**：消亡（~1200 BC 帝國崩潰）

### 1.5 腓尼基宗教
- **起源**：~3000 BC / 黎巴嫩海岸
- **核心信條**：Baal、Astarte 崇拜；影響希臘羅馬
- **狀態**：消亡 / 部分融入迦南

### 1.6 迦南宗教
- **起源**：~3000 BC / 巴勒斯坦
- **核心信條**：El 為主神；Baal 為豐產神；影響早期猶太教
- **狀態**：消亡 / 史料存於舊約

### 1.7 瑣羅亞斯德教（拜火教、祆教）
- **起源**：~1500-600 BC（學界爭議）/ 波斯
- **創始**：查拉圖斯特拉（傳統）
- **核心信條**：善惡二元論（Ahura Mazda vs Angra Mainyu）；火壇聖化
- **核心經典**：阿維斯塔（Avesta）
- **分支**：帕西（印度分支）、伊朗本土
- **當代規模**：~10-20 萬
- **狀態**：活躍但少數

### 1.8 曼達安教（Mandaeism）
- **起源**：~1-3 世紀 AD / 美索或巴勒斯坦
- **核心信條**：施洗者約翰為真先知；重視洗禮
- **核心經典**：Ginza Rabba（寶藏）、Drabsha Qadahiya
- **當代規模**：~6 萬（主要在伊拉克、伊朗、敘利亞）
- **狀態**：活躍 / 受迫害移民族群

### 1.9 撒馬利亞教
- **起源**：~722 BC / 以色列北部
- **核心信條**：基利心山為唯一聖地；五經（僅摩西五經）
- **核心經典**：撒馬利亞五經
- **當代規模**：~800 人（主要在以色列、巴勒斯坦）
- **狀態**：活躍 / 瀕危小族群

### 1.10 德魯茲教
- **起源**：~11 世紀 / 埃及、法蒂瑪王朝
- **創始**：al-Darazi、Hamza ibn Ali（後者為教義奠基）
- **核心信條**：源自伊斯蘭什葉 + 希臘哲學 + 新柏拉圖；輪迴；密意知識
- **核心經典**：Rasā'il al-Ḥikmah（智慧書信集）
- **當代規模**：~100 萬（黎巴嫩、敘利亞、以色列）
- **狀態**：活躍 / 封閉社群

### 1.11 雅茲迪（Yazidi）
- **起源**：~12 世紀 / 美索北部
- **核心信條**：混合瑣羅亞斯德、伊斯蘭、基督教；孔雀天使 Tawûsê Melek 崇拜
- **核心經典**：Kitêba Cilwe（黑書）、Mishefa Reş（白書）
- **當代規模**：~100 萬（主要伊拉克、敘利亞、歐洲流亡）
- **狀態**：活躍 / 多次受迫害

## 2. 古埃及

### 2.1 古埃及宗教
- **起源**：~3100 BC / 尼羅河谷
- **核心信條**：多神（Ra、Osiris、Isis、Anubis、Horus）；法老為神王；來世審判（死者之書）
- **核心經典**：死者之書、棺文、金字塔文
- **狀態**：消亡（~4 世紀 AD 基督教化後消亡）

### 2.2 托勒密 / 希臘羅馬融合宗教
- **起源**：~300 BC - 300 AD
- **核心信條**：Serapis（奧西里斯 + 阿匹斯 + 宙斯融合）
- **狀態**：消亡

## 3. 古典歐洲

### 3.1 古希臘宗教
- **起源**：~1200 BC（邁錫尼）/ 希臘
- **核心信條**：奧林帕斯十二神；英雄崇拜；俄耳甫斯秘儀
- **核心經典**：赫西俄德《神譜》、荷馬《伊利亞德》《奧德賽》
- **狀態**：消亡（~4 世紀 AD 羅馬帝國基督教化）

### 3.2 古羅馬宗教
- **起源**：~753 BC（傳統）/ 羅馬
- **核心信條**：朱比特、密涅瓦、馬爾斯；家神崇拜；皇帝神化（後）
- **狀態**：消亡 / 部分節日融入基督教

### 3.3 北歐宗教（Ásatrú）
- **起源**：~500 BC / 北歐
- **核心信條**：奧丁、托爾、弗雷；世界樹 Yggdrasil；諸神黃昏（Ragnarök）
- **核心經典**：詩體埃達（Edda）、散文埃達
- **當代規模**：現代復興約 2-5 萬（冰島 Ásatrúarfélagið）
- **狀態**：原消亡 / 1970s 後復興

### 3.4 凱爾特宗教
- **起源**：~500 BC / 不列顛、愛爾蘭
- **核心信條**：德魯伊祭司；自然神聖；靈魂轉世
- **核心經典**：無系統經典，威爾斯 Mabinogion、愛爾蘭神話
- **狀態**：原消亡 / 現代復興（Wicca 部分元素）

### 3.5 斯拉夫原始宗教
- **起源**：~500 BC / 東歐
- **核心信條**：多神（Perun、Veles）；自然崇拜
- **核心經典**：原始編年史（Primary Chronicle）記載
- **狀態**：消亡 / 現代復興（Rodnovery 運動）

### 3.6 赫爾墨斯主義（Hermeticism）
- **起源**：~1-3 世紀 AD / 埃及亞歷山大
- **核心信條**：Hermes Trismegistus 傳統；神性內在；宇宙對應
- **核心經典**：Corpus Hermeticum（《赫爾墨斯集》）、Emerald Tablet
- **狀態**：影響文藝復興、玫瑰十字；現代新異教引用

### 3.7 諾斯底教派（Gnosticism）
- **起源**：~1-3 世紀 AD / 地中海
- **核心信條**：真神 vs 造物主（Demiurge）；靈魂來自高位；秘密知識得救
- **核心經典**：拿戈瑪第文庫（1945 發現）
- **狀態**：早期被基督教視異端 / 現代復興（Ecclesia Gnostica）

## 4. 南亞 / 印度

### 4.1 吠陀宗教（婆羅門前身）
- **起源**：~1500 BC / 印度河流域
- **核心信條**：多神、自然神（Indra、Agni）；祭祀（Sacrifice）為核心
- **核心經典**：吠陀本集（4 部：梨俱、沙摩、耶柔、阿闥婆）
- **狀態**：融入印度教

### 4.2 印度教（Hinduism）
- **起源**：演化而成，~6 世紀 BC 形成輪廓 / 印度
- **核心信條**：梵（Brahman）為終極；輪迴（samsara）、業（karma）、解脫（moksha）；多神崇拜 + 一神虔信
- **核心經典**：吠陀、奧義書、薄伽梵歌、往世書、羅摩衍那、摩訶婆羅多
- **分支**：
  - **毗濕奴派**（Vaishnavism）：黑天、羅摩崇拜
  - **濕婆派**（Shaivism）：濕婆為主
  - **Shakti 派**：女神崇拜
  - **Smarta 派**：綜合三神
- **當代規模**：~12 億（印度為主）
- **狀態**：活躍 / 全球擴散

### 4.3 佛教（Buddhism）
- **起源**：~6-5 世紀 BC / 印度（恆河流域）
- **創始**：喬達摩悉達多（Gautama Siddhartha）
- **核心信條**：四聖諦、八正道、緣起、無我（anatta）；涅槃為終極
- **核心經典**：三藏（經律論）；各派有補充經典
- **分支**：
  - **上座部**（Theravāda）：斯里蘭卡、泰國、緬甸、寮國、柬埔寨
  - **大乘**（Mahāyāna）：東亞（中國、日本、韓國、越南）
  - **金剛乘 / 密宗**（Vajrayāna）：藏、蒙古、不丹、尼泊爾
- **當代規模**：~5 億
- **狀態**：活躍

### 4.4 耆那教（Jainism）
- **起源**：~6 世紀 BC（傳統） / 印度
- **創始**：Rishabhadeva（第一位 Tirthankara，傳統）；Mahavira 為第 24 位
- **核心信條**：非暴力（ahimsa）為最高；靈魂獨立於物質；輪迴解脫
- **核心經典**：Angas、Purvas
- **分支**：Digambara（天衣派）、Shvetambara（白衣派）
- **當代規模**：~4-5 百萬
- **狀態**：活躍 / 主要印度

### 4.5 錫克教（Sikhism）
- **起源**：15 世紀 / 印度旁遮普
- **創始**：古魯·那納克（Guru Nanak）
- **核心信條**：一神（Ik Onkar）；平等、反種姓；古魯傳承
- **核心經典**：格蘭特·沙希卜（Adi Granth，後擴充為 Guru Granth Sahib）
- **當代規模**：~3000 萬（印度為主，海外散居）
- **狀態**：活躍

## 5. 東亞 / 中國

### 5.1 儒教 / 儒家（Confucianism）
- **起源**：~6-5 世紀 BC / 中國（魯國）
- **創始**：孔子（孔丘）
- **核心信條**：仁、義、禮、智、信；修身齊家治國；天命觀
- **核心經典**：四書（論語、大學、中庸、孟子）+ 五經（詩、書、禮、易、春秋）
- **當代規模**：文化傳統，無組織信徒（6-10 億文化影響）
- **狀態**：傳統延續 / 是否為宗教爭議（學界：人文主義傳統）

### 5.2 道教（Taoism）
- **起源**：~2 世紀（教團化） / 中國
- **思想源**：老子、莊子（更早）
- **核心信條**：道為終極；無為、自然；內丹 / 外丹；長生
- **核心經典**：道德經、莊子、道藏（明代編 1445）
- **分支**：全真道、龍門派、正一道
- **當代規模**：~1200 萬（教團）；文化影響深
- **狀態**：活躍 / 中國

### 5.3 中國佛教
- **起源**：1 世紀 AD 傳入 / 中國
- **核心信條**：大乘佛教為主；淨土宗、禪宗、天台宗、華嚴宗
- **核心經典**：漢譯大藏經（~2200 卷）
- **當代規模**：~2 億（估，含文化佛教徒）
- **狀態**：活躍

### 5.4 中國民間信仰
- **起源**：上古 / 中國
- **核心信條**：祖先崇拜、神祇體系（玉皇、土地、城隍、媽祖、關公）；風水、命理
- **核心經典**：無固定經典，地方廟志
- **當代規模**：~3-5 億（與佛教、儒難分）
- **狀態**：活躍 / 與制度宗教融合

### 5.5 墨家
- **起源**：~5 世紀 BC / 中國
- **創始**：墨子
- **核心信條**：兼愛、非攻、節用、天志、明鬼
- **狀態**：消亡（漢代後式微）

## 6. 日韓

### 6.1 神道（Shinto）
- **起源**：古代 / 日本
- **核心信條**：萬物有靈（kami）；自然崇拜；祖先；神道無創始者
- **核心經典**：古事記、日本書紀、祝詞（Norito）、風土記
- **分支**：神社神道、神道佛教融合、神道教派（黑住、垂加、神道大教）
- **當代規模**：~1 億（信仰傳統）
- **狀態**：活躍 / 與佛教融合

### 6.2 韓傳佛教
- **起源**：4 世紀 AD 傳入 / 韓國
- **當代規模**：~1 千萬
- **狀態**：活躍 / 與韓國本土信仰融合

### 6.3 日本佛教
- **起源**：6 世紀 AD 傳入 / 日本
- **分支**：天台宗、真言宗（密教）、淨土宗、淨土真宗、臨濟宗、曹洞宗、日蓮宗
- **當代規模**：~9000 萬（含葬祭佛教）
- **狀態**：活躍

## 7. 亞伯拉罕系

### 7.1 猶太教（Judaism）
- **起源**：~2000 BC / 迦南（傳統）
- **創始**：亞伯拉罕（傳統始祖）；摩西頒律法
- **核心信條**：一神（YHWH）；與神立約；律法（Torah）生活；選民觀念
- **核心經典**：塔納克（Torah、Nevi'im、Ketuvim）= 基督教的「舊約」；塔木德（Talmud）
- **分支**：
  - **正統派**（Orthodox）：嚴守律法
  - **保守派**（Conservative）：中間路線
  - **改革派**（Reform）：現代化
  - **重建派**（Reconstructionist）
  - **哈雷迪**（Haredi）：極端正統
  - **哈西迪**（Hasidic）：神秘虔信
- **當代規模**：~1500 萬
- **狀態**：活躍

### 7.2 撒馬利亞教（已列 1.9，重複略）

### 7.3 基督教（Christianity）
- **起源**：~30 AD / 巴勒斯坦
- **創始**：耶穌基督
- **核心信條**：三位一體（聖父、聖子、聖靈）；基督神性 + 人性；救贖、復活；因信稱義（爭議）
- **核心經典**：聖經（舊約 + 新約 27 卷）
- **主要分支**：
  - **天主教**（Catholic）：~12.5 億
  - **東正教**（Orthodox）：~2.2 億
  - **新教**（Protestant）：~8 億
- **當代規模**：~24 億
- **狀態**：活躍

### 7.4 天主教（Catholicism）
- **核心信條**：教皇權威（從彼得傳承）；七聖禮；聖傳與聖經並重
- **當代規模**：~12.5 億
- **狀態**：活躍

### 7.5 東正教（Eastern Orthodoxy）
- **核心信條**：七次大公會議；禮拜傳統；拜占庭為中心；反對教皇無誤
- **當代規模**：~2.2 億（俄羅斯、希臘、東歐為主）
- **狀態**：活躍

### 7.6 新教（Protestantism）
- **起源**：16 世紀宗教改革 / 歐洲
- **創始**：馬丁·路德（路德宗）、加爾文（歸正宗）、慈運理
- **核心信條**：唯獨聖經（Sola Scriptura）、因信稱義、普遍祭司
- **主要派別**：
  - **路德宗**（Lutheran）
  - **歸正宗 / 加爾文宗**（Reformed / Calvinist）
  - **聖公宗**（Anglican）
  - **浸信宗**（Baptist）
  - **長老宗**（Presbyterian）
  - **衛理公會**（Methodist）
  - **五旬節派**（Pentecostal）
  - **福音派**（Evangelical）
  - **靈恩派**（Charismatic）
- **當代規模**：~8 億
- **狀態**：活躍 / 全球擴張最快

### 7.7 摩門教（耶穌基督後期聖徒教會）
- **起源**：1830 / 美國紐約
- **創始**：約瑟·斯密（Joseph Smith）
- **核心信條**：摩門經為另一部經典；現代啟示延續；神人同質；聖殿儀式
- **核心經典**：聖經 + 摩門經 + 教義和聖約 + 無價珍珠
- **當代規模**：~1700 萬
- **狀態**：活躍 / 美國為主

### 7.8 耶和華見證人
- **起源**：1870s / 美國
- **創始**：查爾斯·泰茲·羅素
- **核心信條**：三位一體否認（耶穌為首創受造物）；無地獄（火葬）；拒絕輸血、軍役、慶節
- **核心經典**：聖經（自譯新世界譯本）
- **當代規模**：~870 萬
- **狀態**：活躍

### 7.9 基督科學會（Christian Science）
- **起源**：1879 / 美國
- **創始**：瑪麗·貝克·艾迪
- **核心信條**：疾病為心靈錯誤；祈禱治癒；非醫療介入
- **核心經典**：Science and Health with Key to the Scriptures
- **當代規模**：~10 萬（衰退）
- **狀態**：活躍但衰退

### 7.10 統一教（Unification Church）
- **起源**：1954 / 韓國
- **創始**：文鮮明
- **核心信條**：文鮮明為再臨基督；聖經 + 新經典
- **核心經典**：原理講論
- **當代規模**：~200 萬（含影響圈）
- **狀態**：活躍 / 爭議

### 7.11 伊斯蘭（Islam）
- **起源**：7 世紀 / 阿拉伯
- **創始**：穆罕默德
- **核心信條**：一神（Allah）；穆罕默德為最後先知；服從五功；古蘭經為神聖話語
- **核心經典**：古蘭經、聖訓（Hadith）
- **主要分支**：
  - **遜尼派**（Sunni）：~85-90%
  - **什葉派**（Shia）：~10-15%
- **當代規模**：~19 億
- **狀態**：活躍 / 全球擴張

### 7.12 遜尼派（Sunni Islam）
- **核心信條**：哈里發制度（爭議）；四大法學（哈乃斐、沙斐儀、馬立克、罕百里）；聖訓權威
- **當代規模**：~16 億
- **狀態**：活躍

### 7.13 什葉派（Shia Islam）
- **核心信條**：阿里（穆罕默德堂弟女婿）為正統繼承者；伊瑪目神性（十二伊瑪目派）；追隨十二位伊瑪目
- **分支**：
  - **十二伊瑪目派**（最大）：伊朗為主
  - **伊斯瑪儀派 / 七伊瑪目派**
  - **宰德派**：葉門
- **當代規模**：~2-3 億
- **狀態**：活躍

### 7.14 蘇菲派（Sufism）
- **起源**：~7-9 世紀 / 伊斯蘭世界
- **核心信條**：神秘主義；與神直接合一（fana）；托缽、苦修、詩歌、音樂（dhikr）
- **核心經典**：魯米《瑪斯納維》、阿塔爾《鳥的會議》、伊本·阿拉比《智慧的珠璣》
- **當代規模**：與遜尼什葉融合；獨立組織估 5000 萬
- **狀態**：活躍 / 跨遜尼什葉

### 7.15 阿拉維派（Alawi / Nusayri）
- **起源**：~9 世紀 / 敘利亞
- **核心信條**：阿里神性化；混合希臘哲學、基督教、密特拉；輪迴
- **當代規模**：~300 萬（敘利亞、土耳其）
- **狀態**：活躍 / 受迫害

### 7.16 艾哈邁迪亞（Ahmadiyya）
- **起源**：1889 / 英屬印度
- **創始**：Mirza Ghulam Ahmad
- **核心信條**：Ahmad 為馬赫迪 + 彌賽亞；和平；現代主義
- **當代規模**：~1 千萬
- **狀態**：活躍 / 主流伊斯蘭視異端

### 7.17 巴哈伊信仰（Baha'i Faith）
- **起源**：19 世紀 / 波斯
- **創始**：巴哈歐拉（Mirza Husayn Ali Nuri）
- **核心信條**：上帝唯一、宗教同源、人類一家；各宗教為同一神的不同啟示階段
- **核心經典**：Kitáb-i-Aqdas、Kitáb-i-Íqán
- **當代規模**：~500 萬
- **狀態**：活躍 / 全球散居

## 8. 非洲

### 8.1 非洲傳統宗教（統稱）
- **起源**：上古 / 撒哈拉以南非洲
- **核心信條**：萬物有靈；祖先崇拜；造物主神（不同族群不同名）；巫術、占卜
- **核心經典**：口述傳統，無文字經典
- **當代規模**：~1-2 億（純粹；含混合基督 / 伊斯蘭更多）
- **狀態**：活躍 / 與基督、伊斯蘭融合

### 8.2 約魯巴宗教（Yoruba）
- **起源**：上古 / 奈及利亞、貝南
- **核心信條**：Olodumare 為主神；Orisha 群神；祖先
- **核心經典**：口述傳統
- **當代規模**：~5000 萬（純粹 + 融合）
- **狀態**：活躍 / 經大航海傳至美洲（Santería、Candomblé）

### 8.3 巫毒 / 伏都（Vodun）
- **起源**：18 世紀 / 貝南→海地
- **核心信條**：祖先與精靈（Loa）；祭司（hungan、mambo）
- **核心經典**：口述
- **當代規模**：~6000 萬（西非 + 美洲）
- **狀態**：活躍 / 海地為國教之一

### 8.4 衣索比亞正教（Ethiopian Orthodox Tewahedo）
- **起源**：4 世紀 / 阿克蘇姆
- **核心信條**：基督一性論（非正統二性）；豐富禮拜傳統
- **核心經典**：聖經（81 卷，含 1 Enoch、Jubilees）
- **當代規模**：~5000 萬
- **狀態**：活躍 / 衣索比亞國教

## 9. 大洋洲

### 9.1 澳洲原住民信仰（Aboriginal spirituality）
- **起源**：~6 萬年 / 澳洲
- **核心信條**：夢時代（Dreamtime）神話；歌之徑（Songlines）；祖先創造地形
- **核心經典**：口述、儀式、岩石藝術
- **當代規模**：~70 萬原住民
- **狀態**：活躍 / 部分復興

### 9.2 波里尼西亞傳統（Mana / Tapu 系統）
- **起源**：~1500 BC / 太平洋島嶼
- **核心信條**：Mana（神聖力量）；Tapu（禁忌）；祖先；航海神祇
- **核心經典**：口述、編織圖案
- **當代規模**：~200 萬（含夏威夷、薩摩亞、湯加、毛利等）
- **狀態**：活躍（與基督信仰融合）

### 9.3 毛利信仰（Māori）
- **起源**：~13 世紀 / 紐西蘭
- **核心信條**：Io 為至高神；Mana、Tapu、Whakapapa（系譜）；Haka 為文化儀式
- **核心經典**：口述 + 雕刻
- **當代規模**：~80 萬毛利
- **狀態**：活躍 / 紐西蘭國家文化一部分

### 9.4 夏威夷傳統
- **起源**：~4-7 世紀 / 夏威夷
- **核心信條**：Kū、Lono、Kanaloa 為主神；Hula 為儀式舞蹈
- **狀態**：基督化後式微 / 文化復興中

## 10. 美洲原住民

### 10.1 瑪雅宗教
- **起源**：~2000 BC / 中美洲
- **核心信條**：多神（Itzamná、Kukulkán）；曆法崇拜；球戲；血液儀式
- **核心經典**：波波爾·烏（Popol Vuh，殖民期後寫下）、契倫·巴蘭之書（西班牙征服者記錄）
- **當代規模**：~600 萬瑪雅後裔 + 混合
- **狀態**：原殖民期鎮壓 / 現代復興

### 10.2 阿茲特克宗教
- **起源**：~14 世紀 / 墨西哥中部
- **核心信條**：Huitzilopochtli（戰神太陽神）；人祭；宇宙觀 5 個時代
- **核心經典**：西班牙征服期手稿（Codex Borgia、Codex Borbonicus）
- **狀態**：消亡（16 世紀西班牙征服）

### 10.3 印加宗教
- **起源**：~12 世紀 / 安地斯
- **核心信條**：Inti（太陽神）；Viracocha 創世神；太陽貞女
- **核心經典**：無文字（結繩 quipu）
- **狀態**：消亡（16 世紀西班牙征服）/ 復興運動

### 10.4 北美原住民信仰（統稱）
- **起源**：上古 / 北美
- **核心信條**：萬物有靈；視覺異象；Medicine Wheel；薩滿
- **分支**：平原、森林、西北、沙漠、東南等各族
- **當代規模**：~300 萬
- **狀態**：活躍 / 復興

### 10.5 聖達美（Santa Muerte）
- **起源**：~18-20 世紀 / 墨西哥
- **核心信條**：神聖死亡擬人化；混合天主教 + 民間信仰
- **狀態**：活躍 / 羅馬天主教會反對 / 與毒品暴力掛鉤（部分）

### 10.6 坎迪布雷（Candomblé）
- **起源**：19 世紀 / 巴西
- **核心信條**：源自約魯巴 + 天主教元素；Orixás 崇拜
- **當代規模**：~200 萬
- **狀態**：活躍 / 巴西合法宗教

## 11. 現代新興

### 11.1 山達基 / 科學教（Scientology）
- **起源**：1950s / 美國
- **創始**：L. Ron Hubbard
- **核心信條**：心靈（Thetan）創世；可審計的清流；Xenu 神話
- **核心經典**：Dianetics、Scientology Handbook
- **當代規模**：~10 萬
- **狀態**：活躍 / 爭議（多國視為商業）

### 11.2 雷爾運動（Raelism）
- **起源**：1974 / 法國
- **創始**：Claude Vorilhon（Rael）
- **核心信條**：外星人 Elohim 創造人類；性、愛、冥想
- **核心經典**：「Intelligent Design」書系
- **當代規模**：~10 萬
- **狀態**：活躍 / 爭議

### 11.3 威卡教（Wicca）
- **起源**：1954 / 英國
- **創始**：Gerald Gardner
- **核心信條**：巫術 + 自然崇拜；女神 + 角神；無迫害性 hex
- **核心經典**：Gardner《Witchcraft Today》、Starhawk《The Spiral Dance》
- **當代規模**：~100-300 萬
- **狀態**：活躍

### 11.4 新紀元運動（New Age）
- **起源**：1970-80s / 美西
- **核心信條**：個人靈性、自助、東方元素、占星、靈擺、水晶
- **狀態**：活躍 / 模糊社群
