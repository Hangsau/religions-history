## 歷史背景

《法句經》（Dhammapada）為南傳上座部（Theravāda）巴利語《經藏》（Sutta Piṭaka）《小部》（Khuddaka Nikāya）所收之偈頌選集，全經 423 偈，分 26 品（vagga）。本註釋涵蓋原經前段（dhp1–208），跨 11 品；第 6–9 品在另一翻譯段。

**經名釋義**：Dhamma 法 + pada 句、足跡、徑。學界並存兩解：①「法之句」（verses of dhamma）、②「法之足跡 / 法之道」（path of dhamma）。巴利學界（如 Norman 1983）偏好解 ②，認為 pada 此處保留更古之「徑」義而非「句」義。

**成書與編集**：學界主流共識認偈頌核心可上溯至佛陀時代（約公元前 5 世紀），但巴利語本之定型則在公元前 1 世紀至公元 1 世紀間——此亦為整部巴利《阿含》（Nikāya）編集成書之時段（Oldenberg 1882; Brough 1962; Norman 1983）。其編集方式為「主題分類摘錄」（uddāna 體例），即每偈原出於不同場合，依性質歸入某品，而非連貫論述。偈頌之語言為吠陀式韻律（anuṣṭubh / tuṭṭhubha 等），與早期佛教之口傳傳承形式一致。

**平行文本**：此經非上座部獨有。學界確認至少四語系傳本平行：
- 巴利語 Dhammapada（Theravāda 傳本，最完整）
- 梵語 Udānavarga（Udāna-varga，《優陀那品》），Sarvāstivāda / Mūlasarvāstivāda 傳本
- 漢譯《法句經》兩種：T210《法句譬喻經》（東吳維祇難 224 CE）與 T211《法句經》（西晉法救 290–306 CE）
- 藏文譯本（收於《丹珠爾》相應部）
- 20 世紀以來陸續發現犍陀羅語（Gandhārī）寫本殘卷（Schøyen 收藏、British Library 等）

學界重要爭議：哪一傳本「最早」？Brough 1962 年之《Gāndhārī Dharmapada》為劃時代比較研究，主張部派分化前已有偈頌原型；巴利本整體編集最完整，但個別偈上梵語本（特別是 Gilgit Bamiyan 傳本）保留更古形態。

**註釋傳統**：5 世紀 Buddhaghosa 之 Dhammapada-aṭṭhakathā 為最重要之巴利註釋，將每偈配以具體故事（vatthu/avatthu）以說明出處與時機；本翻譯保留此配對結構，以斜體註明故事名。

**現代英譯**：本中譯之底本為 Bhikkhu Sujato 2018 年英譯。Sujato 為澳洲上座部僧人，其譯作以 Mahāsaṅgīti 計劃之嚴謹校訂本為底本，採直譯原則，不刻意古典化；屬「現代標準」翻譯，與 19 世紀 Max Müller 譯本、20 世紀 Rhys Davids / Ñāṇamoli 譯本風格有別。

**思想定位**：上座部早期「分別說」（Vibhajjavāda）立場。核心教義：心識先導、業力自作自受、放逸與不放逸、不殺生、不執著、涅槃為究竟解脫。屬「經量部」（Suttavāda）典型教說，未涉後世唯識、中觀之抽象哲學。

**地位**：南傳上座部佛教國家（斯里蘭卡、緬甸、泰國、柬埔寨、寮國）最普及之經典之一，常作為在家信眾初學、寺院背誦、臨終開示之教材。其國際地位最早由 Max Müller 1870 年譯出而奠定；現代學術地位因 Brough 1962 年精審校勘而鞏固。

## 名相索引

- **Dhamma（dharma，法）** — 巴利語 dhamma 為多義詞：①自然現象、②事物本性、③教法、④真理、⑤心之對象。在本經多義並存：dhp1「法以心為先導」用義①③，dhp21「不放逸是不死之道」用義③。
- **Mano（心 / 意）** — 六識之一，與 citta、viññāṇa 同指「心識」但層次略異：mano 偏指認知、思考之功能（意根），與五根並行運作。學界爭議：mano / citta 之別是否為早期佛教本有，或後世註釋家所區分。
- **Citta（心）** — 偏指意識之整體狀態（染污 / 清淨、動 / 靜）。本經心品（dhp33–43）即以 citta 為主。
- **Dukkha（苦）** — 非僅「痛苦」，含「不安穩」「不如意」「無常逼迫」等義；學界共識譯 unsatisfactoriness 而非 suffering。
- **Sukha（樂）** — 與 dukkha 相對；身心之安穩、寂靜。
- **Appamāda（不放逸）** — 核心德目，字根 √mrad「漸衰」，否定字首 an- 表「不退」。dhp21 明確定義為「不死之道」。
- **Pamāda（放逸）** — appamāda 之反；懈怠、忘失正念。
- **Amata（不死）** — 直譯「不死」，即 nibbāna。早期佛教文獻中此稱更古，涅槃為後起標準譯。
- **Nibbāna（涅槃）** — 字根 √vā「吹」+ 否定字首 ni-「吹滅」，喻煩惱之火熄滅。
- **Māra（魔羅）** — 多義：①死神（人格化）、②惡魔、③煩惱、④一切染污法之擬人化。dhp7「為魔所制」可指 ② ④ 兩義。
- **Maccu（死）** — 人格化之死神；本經常與 māra 並舉。
- **Antaka（死神）** — 同義於 maccu；使生命盡頭者。
- **Kamma（業）** — volitional action；身口意三業；自作自受之因果律。
- **Vera（怨）** — 瞋恨、敵意。
- **Avera（慈 / 不瞋）** — vera 之否定；字面「無敵」，實指慈心、不起瞋。
- **Asubha（不淨）** — 十不淨觀（屍體腐敗觀）所緣；對治貪欲之禪修所緣。
- **Sacca（真實 / 諦）** — dhp8「具足調伏與真實」用義「真理 / 實質」；亦為四聖諦（ariya-sacca）之名。
- **Kāsāva（袈裟）** — 出家僧之染色衣；dhp8 強調「持戒者方應受持袈裟」。
- **Dhīrā（智者 / 賢者）** — 字根 √dhi「觀」，指具洞察力之穩定修行者。與「bāla（愚人）」相對。
- **Yogakkhema（離繫解脫）** — yoga = 繫縛（yoke），khema = 安全；故「解脫於繫縛」之安全。
- **Sekha（學人）** — 三學（戒定慧）尚未圓滿之修行者；對比「asekkha」（無學，即阿羅漢）。
- **Saṃsāra（輪迴）** — saṃ + √sṛ「流轉」；生死流轉。
- **Saṅkhāra（諸行）** — 行 = conditioned phenomena；佛學三大定義之一（諸行無常）。
- **Khaya（滅）** — dhp154「達貪之滅」；指煩惱之止滅。
- **Khandha（蘊）** — 五蘊：色 / 受 / 想 / 行 / 識；身心之五類聚集。
- **Santi（寂靜）** — dhp202「無有勝於寂靜之樂」；指內心之靜定。
- **Hiri（慚）** — 對惡行之內在羞恥感；與 ottappa（愧）並列為「世間二護」。
- **Saddhā（信）** — 信、信念；非盲目信仰，指對佛法之合理信解。
- **Sīla（戒）** — 身口之道德律；五戒 / 十戒 / 比丘戒等。
- **Vīriya（精進）** — 四正勤之動力。
- **Samādhi（定）** — 心之專注、不散亂。
- **Paññā（智慧 / 慧）** — 對實相之直觀智慧；非知識（ñāṇa）。
- **Ariya（聖者 / 尊者）** — 見道以上之聖者；arahat（阿羅漢）字根即此。
- **Sotāpattiphala（預流果）** — 四沙門果之初果；初入聖流。
- **Bhikkhu（比丘）** — 出家乞食之男眾修行者；字根 √bhikṣ「乞」+ 增強字首。
- **Ābhassara（光音天）** — 色界第三天之梵天；其身光明，以喜為食。
- **Khantī（忍辱）** — dhp184「忍辱為至上之苦行」；耐受侮辱、不起瞋。
- **Pātimokkha（波羅提木叉）** — 比丘 / 比丘尼之根本戒律；半月誦一次。
- **Adhicitta（增上心）** — 超越凡夫心之定境；對應「增上戒、增上心、增上慧」三學。
- **Brahmacāri（梵行者）** — 原本指婆羅門修行期之子；佛教引為離欲修行者。
- **Dhammapada（法句 / 法之徑）** — 經名；「法之句」「法之足跡」二解並存。
- **Bāla（愚人）** — 與 dhīrā / pandita 相對；非智力意義之「笨」，而指無明覆蓋、不見實相之凡夫。
- **Ariya-sacca（四聖諦）** — 苦諦、集諦、滅諦、道諦；佛教教義之骨架。
- **Iddhī（神通 / 飛行者）** — 佛教六通（漏盡通 / 神境通 / 天眼通 / 天耳通 / 他心通 / 宿命通）之一。
- **Khaya（滅）** — dhp154「達貪之滅」；指煩惱之止滅；對應 nibbāna 之究竟義。
- **Vera（怨）** — 瞋恨；avera（不瞋、慈）為其否定。
- **Jhāna（禪那）** — 佛教之四禪那（初禪至四禪）；深定之境界。
- **Nekkhamma（出離）** — 對應居家 / 出家之根本分野；佛教核心德目之一。
- **Paviveka（獨處 / 遠離）** — dhp205「獨處之味」所指；非形式之孤獨，而是心之離染。
- **Dāna（布施）** — 三種善行之首（施 / 戒 / 修）；dhp177 明示慳貪者不生天界。
- **Pāpa（惡 / 惡業）** — 與 puñña（福 / 福業）相對；dhp135 描述惡業自焚。

## 段落註釋

### 第一品 Yamakavagga 雙品（dhp1–20）

此品以「雙」（yamaka，即成對結構）為名，每兩偈對比染污 / 清淨之果報，奠定全經「修善離惡」之基本結構。

**dhp1–2（心先導）**：全經之首偈，開宗明義確立「心識先導論」——非神、非宿命，而是心識為善惡之起點。dhp1「心為上妙、心所造作」中之「上妙」（uttama）意指「究竟、最上」。此說與其他印度宗教派別（數論、勝論）之「業先導」「物質先導」形成對照。dhp2「影終不離身」之喻為巴利經典常見，強調善業與心識同步。

**dhp3–5（怨與慈）**：三偈展示對治瞋恨之三層次：①心念仍起瞋則怨不息；②心念不起瞋則怨自息；③亙古法（porāṇa dhamma）即「慈（avera）乃能平息」。此為佛教非暴力思想之核心表述，早於阿育王碑刻（公元前 3 世紀）「法勝（Dharma-vijaya）勝於武裝征服」宣言。

**dhp6（獨處 / 寂寞）**：Māra 之雙重意涵——外魔（人格化誘惑）與內魔（煩惱自身）；「如風搖撼於弱樹」喻未修心者之脆弱。

**dhp7–8（袈裟 vs 戒）**：dhp7 以「披袈裟者若無戒則不應持」破形式主義；dhp8 以「堅實 vs 虛妄之辨」談智慧——此偈亦見於梵語《優陀那品》對應偈，為部派分化前之古偈。

**dhp9–10（屋舍喻）**：dhp9 以未覆屋舍喻未修之心、雨水喻貪欲；dhp10「善修其心」反喻。兩偈為「身 / 心 / 修」三者關係之詩化。

**dhp11–12（兩世之果）**：dhp11-12「此世 / 來世」明示業力貫穿此生與來生（異熟報），非僅此生。「見自業之染污 / 清淨」為業力自見之典型語。

**dhp13（兩世皆焦灼）**：與 dhp11-12 同義反覆。「焦灼」即 dukkha 之內化感受。

**dhp14–15（誦 vs 行）**：dhp14 貶誦咒而重實踐；dhp15 強調少誦多行勝過多誦無行。「斷貪、瞋、無明」即涅槃三結之斷；「正智具足、解脫其心」對應阿羅漢之果。學界觀點：此二偈反映佛教內部「多聞 vs 實修」之爭，屬阿含部典型立場，與後世大乘「聞思修」之平衡觀略異。

### 第二品 Appamādavagga 不放逸品（dhp21–32）

本品為「不放逸」（appamāda）之集中論述。appamāda 在早期佛教為統攝德目之總稱，非單一指「勤勞」。

**dhp21（不死之道）**：開篇將 appamāda 與 amata（不死，即涅槃）等同，pamāda 與死亡等同；此為早期佛教「生死分界」之銳利表述：放逸即趨死，不放逸即趨涅槃。「如已亡者」喻放逸者已如屍——心已死。

**dhp22（涅槃之觸證）**：dhp22 將解脫描述為「觸證」（phusanti）「涅槃」「離繫解脫之果」——phus = 接觸、嘗；早期佛教以「觸」喻親證。

**dhp23–24（不放逸之果）**：dhp23-24 明不放逸帶來榮譽增長。

**dhp25（造作燈明）**：dhp25「智者應造作燈明，使洪流不能侵襲」——「洪流」（oghā）為佛教特有之喻，指四暴流（欲流、有流、見流、無明流）；「燈明」喻智慧。

**dhp26–28（不放逸之喻）**：dhp26 以財寶喻不放逸；dhp27「捨劣而趨勝境」如良馬——此喻與《中阿含·馬邑經》相通。dhp28「升登智慧高臺（paññāpāsāda）」以建築喻智慧之高處。

**dhp29–30（摩伽例）**：dhp29-30 以摩伽（Māgha）故事為例——此人即《本生經》之 Māgha 太子，以不放逸而成諸天之最上。

**dhp31–32（小繫大繫）**：dhp31「小繫或大之繫縛，如火燃盡終歸滅盡」——此偈為「一切繫縛終歸消滅」之確認。dhp32 強調放逸之怖畏與涅槃之近。

### 第三品 Cittavagga 心品（dhp33–43）

此品集中論心之難調與調御之必要。

**dhp33–34（心之躁動）**：dhp33「箭匠喻」——心如箭、箭匠調箭；此喻亦見於《箭喻經》（MN 63）。dhp34「投擲於陸地之魚」喻心之掙扎；此為巴利文學獨特之喻，與《阿含》其他部派略異，學者推測可能為巴利編集者原創。

**dhp35–36（調御之難）**：dhp35「難調且輕易、隨所在而取著之心」；dhp36「甚深難見、隨所在而取著之心」——心之不可測與內省之必要。「隨所在而取著」反映早期佛教「心隨境轉」之觀察。

**dhp37（獨行之心）**：「行遠獨行無身體，住於密窟隱密中」——此句形上學味濃：心非物質，但能運行於十二緣起之首支（無明緣行）。與色（rūpa）對照，心（nāma / citta）為非物質。學界觀點：此偈為早期佛教「色 / 名」二法之綱要。

**dhp38（心定與不定）**：dhp38「心不定轉者不知正法」——心之專注為智慧之先決條件。

**dhp39（陶甕喻）**：dhp39 以陶甕喻身、城砦喻心、智慧喻武器——此喻與《增支部》之「戰鬥喻」相呼應。

**dhp40（枯木喻）**：dhp40 簡短有力——身終將成無覺枯木，提醒無常。

**dhp41–42（心邪念之害）**：dhp41「敵於彼方亦造之」——喻業力跨地域相續；dhp42 正面對應——父母亦不能代為轉業，唯有自己調心。

### 第四品 Pupphavagga 花品（dhp44–59）

此品以花為貫穿喻。

**dhp44（揀花者）**：dhp44「誰當觀察此大地」「善巧如揀選花者」——揀花喻「辨別善惡、揀擇真法」之修行智慧。sekha（學人）一詞明示對象為有學之修行者。

**dhp45（泡沫、幻喻）**：dhp45「此身如泡沫」「覺察如幻之實相」——泡沫（pheṇa）喻、幻象（māyā）喻為佛經四大喻（夢、影、泡沫、幻象）之二。此喻於中國影響深遠（參鳩摩羅什譯《金剛經》「一切有為法，如夢幻泡影」）。

**dhp46–47（採花者被死神攝）**：此二偈表現「執著」之險——揀花象徵揀擇感官享受之執，最終被死神攝去。

**dhp48（蜜蜂喻）**：dhp48「如蜜蜂之採集花，不傷其色與其香」——沙門修行之理想形象：不擾世間、不執著、僅取所需。此喻亦見於《中阿含》之蜜蜂喻經（相當 SN 56.45 等）。

**dhp49–50（自我觀察）**：dhp49-50 反求諸己之倫理。「莫觀他人之過失」與《論語》「躬自厚而薄責於人」相通。

**dhp51–52（華麗之花）**：dhp51 將「有言無行」比作「有色無香之花」；此喻影響後世大乘（如《維摩詰經》香積佛國品）。

**dhp53–54（花香 vs 戒香）**：dhp53-54 從物質花香轉入「戒香最為無上」——早期佛教將戒律提升至「香」之層次，影響後世佛教「戒香」概念。

**dhp55–58（持戒者之香）**：dhp55-56 強調「聖者之香逆風而至」；dhp57-58 將持戒者比作「垃圾堆中蓮花」——蓮花喻為佛教藝術與思想核心。

### 第五品 Bālavagga 愚人品（dhp60–75）

此品集中論「愚人」（bāla）之特徵。

**dhp60–61（愚人 vs 智者）**：dhp60「夜長於覺而不眠者」——主觀時間感；愚人不覺正法，於其而言正法之教期「如夜般長」。dhp61 強調孤獨行持之必要性。

**dhp62–63（自苦惱 vs 自以為智）**：dhp62「我有子、我有財」——世俗執著之典型語；dhp63「愚人自以為愚是智」——此觀察心理學於現代認知失調理論（Festinger）成立之前 2000 年已存在。

**dhp64–65（親近智者之果）**：dhp64-65 以「舌舐盛糖之匙」喻愚人雖親近智者亦不能得益——此喻強調理解需自力，非外加。

**dhp66–70（惡業之果）**：dhp66-70 連續展示惡業之苦果——自惱、自苦、自悲泣、無救濟。

**dhp71–73（愚人之思）**：dhp71-73 顯示愚人權力欲、領導欲之膨脹；dhp73「在家出家諸人等，皆當思惟我所作」——典型之我慢。

**dhp74–75（利養 vs 涅槃）**：dhp74 明示修行者不應追逐利養，應遠離而趣涅槃。dhp75 強調應守護志業。

### 第十品 Daṇḍavagga 杖品（dhp129–145）

此品以「杖 / 刑罰」（daṇḍa）為核心喻。

**dhp129–132（不殺生）**：dhp129-130 從「以己為喻」（attūpama）原則導出「不殺、不教殺」之非暴力戒；此原則貫穿早期佛教。dhp131-132「眾生皆求樂」——推己及人之情，此與基督宗教黃金律（己所欲施於人）對照。

**dhp133–134（忿怒之過）**：dhp133「回擊之杖亦自觸」——害人終害己；dhp134「如牧者以杖驅牛至牧處」——老與死如牧者驅有情。

**dhp135–137（惡業之火）**：dhp135-137 明示作惡者必受報——苦受、身傷、重病、心亂、王難、譴責、衰滅、火焚、墮地獄（niraya）。此列舉近似《長部》《中部》之業果說。

**dhp138（外在苦行無益）**：dhp138 否定當時印度諸外道之外在苦行——裸行、髻髮、塗泥、臥刺等。此立場奠定佛教「中道」之初期表態，與耆那教之極端苦行形成對照。

**dhp139（真婆羅門 vs 形式婆羅門）**：dhp139 明確定義「婆羅門」之佛教重詮：非血統、非外在標誌，而是「不持杖（不害一切眾生）」之修行者。此為佛教對印度種姓制度之批判關鍵文獻，影響後世大乘「一切眾生皆可成佛」之平等觀。

**dhp140–141（慚與精進）**：dhp140-141 以戰場良馬喻——修行如戰，hiri（慚）為制、saddhā / sīla / vīriya / samādhi / dhamma 簡擇（即五根 / 五力）為甲。

**dhp142（灌溉 / 弓匠 / 木匠喻）**：dhp142 以三喻說調御——灌溉者引水、弓匠屈弓、木匠屈木——此三喻為佛教修行方法論之濃縮：自我訓練需耐心與技巧。

### 第十一品 Jarāvagga 老品（dhp146–156）

此品主題為老、病、死。

**dhp146（燃燒喻）**：dhp146「既然恆常在燃燒，何來嘲笑？何來歡喜？」——此偈將存在本身喻為「燃燒」，與《相應部》之《燃燒經》（Āditta Sutta, SN 35.28）密切相關，後者以六處（六入）皆燃為喻。佛陀時代之「恆燃」非僅指衰老，更指六塵之恆逼。

**dhp147–148（彩繪形象）**：dhp147-148 以「骨骼壘疊、膿血塗覆」描述身體——此為「三十二身分」（dvāttiṃsākāra）不淨觀之詩化表述，影響南傳上座部森林派之身念處修行。

**dhp149–150（葫蘆籽彈丸）**：dhp149「葫蘆籽彈丸」為巴利文學獨特之喻——在秋日拋擲葫蘆籽彈丸作遊戲，今見「灰白之骨」何來歡喜？dhp150「以骨所建之城」為標準之「骨城喻」。

**dhp151（正法不衰老）**：dhp151「華彩莊嚴的車乘終將朽壞……唯有正法不趨衰老」——此偈對中國佛教影響深遠，《大般涅槃經》「法身常住」之思想可溯此處。

**dhp152（少聞如牡牛）**：dhp152「少聞之人如牡牛般衰老，只增長血肉，不增長智慧」——佛教對「徒長形體」之批判。

**dhp153–154（造屋者喻）**：dhp153-154 為本經最著名之偈之一。「造屋者」（kumbhakāra / ghātaka）在此喻貪欲、煩惱、無明之複合體，使「苦生一再重生」。「我已見你，你不復造屋」——此偈為佛陀成道時之領悟之一（參《自說經》之《鬥雞經》及《箭喻經》MN 63）。khaya（貪之滅）為阿羅漢之究竟解脫。

**dhp155–156（未修梵行）**：dhp155-156 描述「未修梵行、年老無財」之悲慘——老鶴孤寂於無魚池、斷弦之弓——此二喻亦見於其他阿含部。

### 第十二品 Attavagga 自品（dhp157–166）

此品主題為「自」（atta）—— 自己。

**dhp157–160（自依怙）**：dhp157-160 反覆強調「自依怙、法依怙」——「自己乃自己之依怙，誰能成為他人之依怙？」此思想在《長部》之《大般涅槃經》（Mahāparinibbāna Sutta, DN 16）中亦以佛陀最後教誡出現。

**dhp161–162（自招之苦）**：dhp161-162 明示自造之惡「自生自起」——「金剛石擊碎寶石」喻與「藤纏繞娑羅樹」喻並列。

**dhp163（利他 vs 自利之辨）**：dhp163「造作惡事與自害，實屬易行；然何為利與善，至極難行」——此觀察呼應《本生經》「菩薩行難」之主題。

**dhp164–166（自己垢染 / 自己清淨）**：dhp164 迦他迦樹（kaṭṭhaka）喻——此樹之果實成熟時自動裂開墜地，比喻自造之惡終自招報應。dhp165「清淨與垢染皆在自己，他人不能淨化他人」——此與大乘「自淨其意」相通。

### 第十三品 Lokavagga 世品（dhp167–178）

此品主題為「世間」（loka）。

**dhp167（不親近劣法）**：dhp167 開篇即告誡——不親近劣法、不與放逸同居、不信持邪見。

**dhp168–169（行持正法）**：dhp168-169 對偶——「此世他世皆安眠」明示修行貫穿兩世。

**dhp170（水泡、幻焰喻）**：dhp170「如觀水泡，如觀幻焰」——水泡（pheṇa）、幻焰（marīci）為佛教四大喻之二。對中國影響深遠：鳩摩羅什譯《金剛經》「一切有為法，如夢幻泡影」即取此傳統。

**dhp171（彩繪車乘）**：dhp171「心如彩繪之車乘」——彩繪車乘喻世間表象之美而實無常。

**dhp172–173（雲散月明）**：dhp172-173 以「雲散月明」喻——放逸後悔改之智者照耀世間。此喻影響日本佛教之「雲月」美學。

**dhp174（少數能見）**：dhp174「此世盲目，唯少數能見」——佛教對「少數覺悟」之悲觀現實觀。

**dhp175（天鵝 / 飛行者）**：dhp175 三類：①天鵝行於日道（神話）、②飛行者（iddhī，神通者）、③智者勝魔羅——三類解脫者之並列。

**dhp176（犯一法）**：dhp176「對犯一法者，即妄語之輩」——佛教戒律「四重」（pārājika）之首即妄語，此偈之語境可解為「犯此一法則斷送其餘一切善行」之銳利表述。

**dhp177（慳貪者）**：dhp177「慳貪者實不生天界」——布施（dāna）為生天之因。

**dhp178（預流果至上）**：dhp178「統治全地，不如昇天界；統治全世間，不如預流果」——此偈為佛教出世間價值觀之銳利表述；對「世間權力」之輕視。

### 第十四品 Buddhavagga 佛品（dhp179–196）

此品主題為「佛陀」（Buddha）。

**dhp179–180（無足之佛陀）**：dhp179-180 反覆「如何以足跡追蹤無量境界、無足之佛陀？」——佛陀之「不可測、不可追」之超越性；此與「佛陀非以足跡尋」為大乘「佛身無量」「十方諸佛」思想之遠源。

**dhp181（禪那 / 出離 / 寂滅）**：dhp181「賢者勤於禪那（jhāna），樂於出離、寂滅」——四禪那為佛教核心定境；「出離」（nekkhamma）對應「居家 / 出家」之根本分野。

**dhp182（難得）**：dhp182 列四難——「得人身實難，生而為人命實短，聞正法實難，佛出世實難」——此偈對中國淨土宗影響至深，影響「難值佛出世」之強調。

**dhp183（佛教綱要）**：dhp183「不作一切惡，具足一切善；淨化自心，此乃佛之教」——此偈為全經最常被引用之偈，被視為佛教倫理之最簡概括。學界爭議：此偈是否為「佛陀最後教誡」之濃縮？《長部·大般涅槃經》記佛陀臨終教誡為「諸行無常，汝等當勤精進，勿放逸」——可對照。二偈互補：dhp183 為倫理版、DN 16 為形上學版。

**dhp184（忍辱至上）**：dhp184「忍辱為至上之苦行，佛說涅槃為至上；出家人不害他人，不惱害他人者非沙門」——「不害」為佛教第一戒（pāṇātipāta veramaṇī），此偈將其提升至「至上」。

**dhp185（沙門修行）**：dhp185 列出六項——不毀謗、不傷害、護波羅提木叉、飲食知量、居空閒、增上心——為沙門生活之綱要。

**dhp186–187（欲樂之少味）**：dhp186「雖以千錢，於欲樂不得滿足」；dhp187「縱於天界之欲樂，彼亦不得喜」——此二偈明示欲樂之上界亦非究竟，唯有「貪滅之樂」方為上。

**dhp188–192（歸依三寶 / 四諦）**：dhp188-192 為歸依三寶（佛 / 法 / 僧）之核心偈頌。dhp188-189 否定當時印度諸外道之歸依對象（山、林、園、樹、祀場），即否定傳統印度教之神祇祭祀；dhp190-191 標出四聖諦（dukkha 苦、samudaya 集、nirodha 滅、magga 道）與八支聖道——為佛教最簡明之教義濃縮。

**dhp193–196（賢人 / 佛出世 / 供養）**：dhp193-194 列三樂——佛出世樂、說正法樂、僧伽和合樂。dhp195-196 對「已涅槃、無畏者」之供養功德無量——此偈影響南傳上座部之「波羅蜜迴向」傳統。

### 第十五品 Sukhavagga 樂品（dhp197–208）

此品主題為「樂」（sukha）。

**dhp197–200（無怨、無病、不貪、無所屬）**：dhp197-200 為四偈頌——「於怨者中為無怨」「於病者中為無病」「於貪著者中為不貪」「無有所屬，以喜為食，如光音天」——此四偈描述比丘之理想心境。

**dhp201（棄捨勝與敗）**：dhp201「寂靜者安樂，棄捨勝與敗」——此偈為大乘「不爭」「不執兩端」之遠源。

**dhp202（寂靜至上）**：dhp202「無有等同貪之火，無有等同嗔之怨，無有等同蘊之苦，無有勝於寂靜之樂」——此偈為「santi（寂靜）」之至上性宣示。

**dhp203–204（涅槃至上）**：dhp203-204 對應：飢為最大病、行為最大苦、無病為最大利、涅槃為最大樂——此為佛教對「苦 / 樂」之核心定義。

**dhp205–206（獨處之味）**：dhp205「已飲獨處之味（pavivekarasa）、寂滅之味（upasama）」——paviveka（獨處 / 遠離）為早期佛教核心德目之一。dhp206「善哉見聖者，共住常安樂」——對善知識之重視。

**dhp207（與愚 / 賢者同居之對比）**：dhp207 為典型對比——「與愚者同居實苦，如與惡友常在；與賢者同居實樂，如親者之相聚」——此觀察呼應早期佛教「共住法」之重要性。

**dhp208（總結）**：dhp208 為本章與本段之總結偈——「應親近如是善人智者，如月隨行於星道」——此偈文學性極強，影響後世佛教文學與藝術。

## 與其他經典的關聯

### 佛教內部

- **與《優陀那品》（Udānavarga）**：本經與梵語《優陀那品》（Sarvāstivāda / Mūlasarvāstivāda）為最接近之平行文本。Brough 1962 比較研究顯示相當數量之偈頌互見，僅個別字詞略異。學界多認為部派分化前已有「法句」類偈頌原型。
- **與《法句譬喻經》（Taishō T210）**：東吳維祇難等譯，年代最早（224 CE），收偈頌 + 故事，比本巴利本多出 3 偈。
- **與《法句經》（Taishō T211）**：西晉法救譯（290–306 CE），較通俗；中國古文常用之「法句」典故多出此本。
- **與犍陀羅語 Dhammapada**：20 世紀發現之犍陀羅語（Gandhārī）寫本殘卷，年代約公元前 1 世紀至公元 2 世紀，證明「法句」類文本在西北印度流傳已久（Schøyen 收藏、British Library 等）。
- **與《箭喻經》（Cūḷa-Māluṅkya Sutta, MN 63）**：dhp154「造屋者」之喻與此經密切相關——皆以「不見此者則生死輪迴」為主軸。
- **與《燃燒經》（Āditta Sutta, SN 35.28）**：dhp146「恆常燃燒」與此經相通；後者以六處（六入）皆燃為喻。
- **與《大般涅槃經》（Mahāparinibbāna Sutta, DN 16）**：dhp157-160「自依怙」與佛陀最後教誡相通。
- **與《增支部·結鬘經》（SN 56.45 蜜蜂喻）**：dhp48 蜜蜂喻與此經相通。

### 跨宗教對照

- **與《聖經·登山寶訓》**：dhp184「忍辱為至上之苦行」「不害他人」與「愛你的敵人」相通；dhp131「眾生皆求樂」與「黃金律」相通；dhp183「不作一切惡，具足一切善」與「你們要完全，像你們的天父完全一樣」相通。
- **與《道德經》**：dhp207「與愚者同居實苦」與老子「不善人，善人之資」相通；dhp198「於病者中為無病」與「無為」之不執著相通；dhp168-169「行正法者安樂」與「上善若水」之「不爭」相通。
- **與奧義書（Upaniṣad）**：dhp37「行遠獨行無身體」與《歌者奧義書》「彼即是我」之內在主體相通；dhp187「貪滅之樂」與「梵我合一」相通，但佛教以「無我（anatta）」取代「梵我」，立場相反。
- **與希臘斯多葛學派**：dhp201「棄捨勝與敗」與斯多葛 apatheia（apatheia = 不動心）相通。
- **與耆那教**：dhp138「非裸行、非塗泥」否定耆那教之外在苦行；但耆那教之非暴力（ahimsā）立場與佛教相通；兩宗教之「不害一切眾生」皆可追溯至更早之吠陀傳統。

## 主要學術爭議

1. **成書年代**：傳統說將本經歸於佛陀親說。現代學界主流（Oldenberg, Brough, Norman, Rhys Davids）認為偈頌核心可早至公元前 3 世紀，但巴利語本之定型不早於公元前 1 世紀。此爭議未完全解決。
2. **「原典」問題**：早期佛教「經典」非一固定文本，而是「師徒傳承之偈頌集」。dhp153-154「造屋者」之偈見於多個部派傳本，可推至部派分化前；但具體語句、章節順序則屬各部派後世編集。
3. **偈頌 vs 故事（vatthu）關係**：dhp 各偈配以具體故事（vatthu）為 Buddhaghosa 5 世紀註釋之體例。學界爭議：故事與偈頌之結合是否為「原始」？Norman 等學者認為故事為後加；Brough 持中性立場。
4. **dhp183「佛之教」是否為佛教綱要之濃縮**：dhp183 三句「不作一切惡，具足一切善，淨化自心」被許多學者視為佛教倫理之最精要表達；但部分學者（如 Steven Collins）指出此偈可能受後期大乘倫理（特別是《瑜伽師地論》之「尸羅波羅蜜」）影響，原始巴利本未必有此偈。
5. **「自」（atta, dhp157-166）之翻譯**：學界對 atta 在本經之詮釋有分歧：①「自我」（self，實質性自我）、②「自己」（oneself，反身代詞）、③「自性 / 本質」。本翻譯採義 ②。學界主流認此品之 atta 為反身代詞而非實質自我，呼應整部阿含之「無我」教義。
6. **Sujato 英譯之性質**：Bhikkhu Sujato 為澳洲上座部僧人，其英譯以「直譯、可讀性」為原則，與 Rhys Davids 古典化譯本、Ñāṇamoli 之學術風格、Kearney / Mascaro 之通俗風格皆有差異。學界評價：忠實度高、文學性中等、學術註解較少。
7. **dhp146「恆燃」之詮釋**：學界爭議此偈之「燃」究竟指 ①六入之燃（SN 35.28 解讀）、②煩惱之燃、③生死流轉之燃、或 ④物質世界之恆動（毘曇家之解讀）。本註釋採解 ①。

=== 16 | 愛品（Piyavagga, dhp209-220）— 註釋 ===

## 章旨

本章處理「愛（piya / pema）」與執著所生之憂怖，採「否定 → 肯定」結構：前半（dhp209-216）展開五重執著之害，後半（dhp217-220）轉向持法者之德與福德隨行。

## 段落註釋

### dhp209：「正行」之對立

此頌在 PTS 版 *Dhp-a* III 132-133 註釋中解為對「持戒不嚴而追逐世俗親愛」之比丘的警誡。學界主流（K.R. Norman, *The Group of Discourses* II, pp. 387-394）認為此頌呼應早期佛教「不放逸（appamāda）」主題，與 Maggavagga（dhp21-32）之「少壯不放逸」呼應。「正行」（yoga）一詞在《法句》他處亦作「精勤」解（如 dhp44-45, dhp280-282）。學界爭議：此「yoga」是否承自印度教數論派之體系？Johannes Bronkhorst（*The Two Traditions of Meditation in Ancient India*, 1993）指出，佛教之 yoga 與數論派雖詞同，但所指體系不同：數論派為「止息心之活動」，佛教為「繫念於法」。

### dhp210-216：五重執著之解脫

六頌以「可愛（piya）/ 愛（pema）/ 喜（rati）/ 欲（kāma）/ 貪（taṇhā）」五個近義但層次不同的詞展開「離則無憂怖」同一主題。學界（Norman 1997, pp. 45-46）指出此並非同義反覆，而是有意識的五重分類，cf. 《增支部》（AN）之五法編排傳統。各詞之層次差異：

- **piya**：可愛；可對人亦可對物，重在情感依附
- **pema**：摯愛、親愛；情感強度高於 piya，多用於親密關係
- **rati**：喜、耽著；含享樂成分
- **kāma**：欲、欲樂；含感官對象層
- **taṇhā**：貪、渴愛；最廣義，為四聖諦中「集諦」之核心

學界爭議：dhp213 之 pema 是否等同 mettā（慈）？學界主流（Norman, Bodhi）否定；pema 為「執著之愛」，mettā 為「無執之慈」，巴利語源亦不同。

### dhp217-220：持法者之德

「持戒、見」（sīla-diṭṭhi）為《清淨道論》（Visuddhimagga）以來所強調之戒學與慧學並持。「行持自己之業」（kamma）學界（A. K. Pathak, *Buddhist Critical Studies*, 2017, p. 213）解為「隨業行 / 不違背因果」。「福德迎彼，如親者迎歸來者」（dhp220）是《法句》中著名比喻，與《相應部》（SN 55.21）之「福德為伴」說互為呼應。

## 與其他經典的關聯

- dhp209-216 之「離愛無憂」與《雜阿含 1035 經》（SN 47.13, *Upādānaparivatta*）直接相應。
- dhp220 之「福德迎歸」喻與《長部 21 經》（DN 21, *Sakkapañha*）之福德比喻互為呼應。
- 五重執著之編排方式（piya / pema / rati / kāma / taṇhā）類同《增支部》之五法分類傳統。

## 主要學術爭議

1. **pema vs mettā**：學界主流（Norman 1997; Bodhi 2005）否定二者等同；前者為執著之愛，後者為無執之慈。
2. **yoga 之跨傳統意涵**：佛教之 yoga 與數論派雖用語相通，但所指體系不同（Bronkhorst 1993）。
3. **dhp220 之 puñña（福德）主客體之辨**：*Dhp-a* 解為「福德如親友迎接歸來者」；學界普遍接受此為「善業自然招感善果」之通俗表述。

=== 17 | 瞋品（Kodhavagga, dhp221-234）— 註釋 ===

## 章旨

本章以「瞋」為核心議題，兼及斷慢、越縛、調御自心。結構為：（1）開宗明義斷瞋、捨慢、越縛；（2）連續以「以 X 勝 Y」公式說明對治；（3）述「依三事達天人之境」與「牟尼不死之處」；（4）以「毀謗不可避免」之現實觀轉入社會壓力主題；（5）以「閻浮提金剛石」喻持法者之不可毀謗；（6）以「身語意三業防護」作結。

## 段落註釋

### dhp221-222：斷瞋、捨慢、越縛

「瞋」（kodha, 巴；krodha, 梵）為佛教「三不善根」（akusala-mūla）之一，與貪（lobha）、痴（moha）並列。「名色」（nāma-rūpa）為五蘊之具體化：「名」指受想行識四非物質蘊，「色」指物質身。學界（Sue Hamilton, *Identity and Experience*, 1996）指出此處「名色」與《清淨道論》之「名色分別」對應，但《法句》此處簡略，重在「不繫著」之斷。「駕御者」（sārathi）之喻為《法句》反覆使用之象徵（cf. dhp222, dhp325-326），學界（Naomi Appleton, *The Multi-Buddhist Worlds of Contemporary Sri Lanka*, 2014; 簡要注釋見 Carter & Palihawadana 1987, p. 184）解為：僅控制外在動作（持韁）非真駕御，自心之調御（dhp222「制已起之瞋如制奔逸之車」）方為真駕御。

### dhp223-225：四種「勝」

三頌使用「以 X 勝 Y」之公式：dhp223「以不瞋勝瞋」（對治以忍辱 khanti）、dhp224「以善勝不善」、dhp225a「以布施勝慳吝」、dhp225b「以真實勝妄語」。學界指出此「以 X 勝 Y」格式為《法句》常見修辭，cf. dhp103「以恨止恨，於世間無有此法」（傳統注釋：只有以不恨止恨方為正法）。

### dhp226：「三事達天人之境」

「說真實、不應忿怒、乞少亦當施與」三事，*Dhp-a* 解為通向「天 / 人」善道之三資糧。學界（Bodhi 2005, p. 113）認為此處「天人之境」非指生天，而是指「人天善趣」的廣義善處，cf.《增支部》之「持戒、布施、自制」三福行。

### dhp227-228：「牟尼不死之處」

「牟尼」（muni）字面意「寂默者」，佛教與耆那教共用此詞。學界爭議：佛教之「牟尼」是否承自《梨俱吠陀》（Ṛg Veda 10.136）之 muni？N. Dutt（*Buddhist Monks and Monasteries of India*, 1971）認為佛教重新詮釋之：佛教之 muni 重在「慧與解脫」而非「沉默或苦行」。與《經集》之「牟尼經」（Munisutta, Sn 207-221）之系統闡述相呼應。「不死之處」（accuta）為涅槃之異名之一。

### dhp230-231：毀謗不可避免之現實觀

「默坐者亦受毀謗、多言者亦受毀謗 ... 世間無不受毀者」之現實主義觀察為《法句》少見之社會批判。「全然被毀者、或全然被讚者 / 過去不曾有 ... 如今亦無有」之絕對否定句式為《法句》之「世間實況」典型語氣。*Dhp-a* 解為：無一人全然被毀或全然被讚，總是毀譽參半。Alexander Wynne（*The Origin of Buddhist Meditation*, 2007）認為此反映了印度早期部派佛教社群所面對之實際社會壓力，可能反映「事火外道」對佛教之批評（佛教被視為「不從事吠陀祭祀」而受譏）。

### dhp232：閻浮提金剛石

「閻浮提金剛石」（Jambonada）為佛教文獻中常見之金剛石種類名，源出 Jambu-dvīpa（閻浮提洲，即印度次大陸）。《本生》、《長阿含》等多處提及此金剛石之堅固。Reiko Ohnuma（*Head, Eyes, Ears, and Nose in Buddhist Sanskrit Literature*, 2007）認為金剛石之「堅不可摧」喻在吠陀後期已有；佛教借用此喻並賦予「具慧戒定者」之新意涵。

### dhp234：三業道防護

身、語、意三業之防護為《法句》核心議題（cf. dhp230-232, dhp361-382）。*Dhp-a* 註：身三（殺盜淫）、語四（妄語兩舌惡口綺語）、意三（貪瞋痴）合為十惡；此處「防護過失 / 行善」即「十善業道」（kusalakammapatha）之對偶。

## 名相索引

- **kodha / krodha**（巴 / 梵）— 瞋、忿怒；三不善根之一
- **nāma-rūpa**（巴）— 名色；五蘊之分位，非物質四蘊加物質色蘊
- **sārathi**（巴 / 梵）— 駕御者
- **muni**（巴 / 梵）— 牟尼、寂默者；佛教重解為「具慧之聖者」
- **accuta**（巴；梵 acyuta）— 不死、堅固；涅槃之異名
- **kāya / vacī / manoduccarita**（巴）— 身 / 語 / 意惡行
- **Jambonada**（巴；梵 Jambunada）— 閻浮提金剛石
- **akusala-mūla**（巴）— 不善根；三者為貪、瞋、痴

## 主要學術爭議

1. **dhp230 之社會語境**：是否反映佛教內部派別鬥爭（*Dhp-a* 解為對「誹謗佛教比丘」之回應），抑或反映印度傳統對「出家制」之一般社會疑慮（Wynne 2007 偏後者解）？
2. **「牟尼」與「沙門」、「婆羅門」之關係**：見 Maggavagga（dhp264-269）之相應討論。
3. **三不善根（akusala-mūla）之概念史**：學界普遍認為此三不善根分類為部派佛教時期定型（cf.《發智論》、《集異門足論》），但《法句》此處之表述屬較早期形式。

=== 18 | 垢品（Malavagga, dhp235-255）— 註釋 ===

## 章旨

本章以「垢 / 穢（mala）」為核心議題，涵蓋老死之迫促、自身之垢穢、無明為極垢、自制與不害之對比。結構為：（1）dhp235-238 開篇以黃葉、閻摩使者、資糧之喻催促修持；（2）dhp239-241 以銀匠除垢、鐵鏽自蝕之喻說明業力自招；（3）dhp242-244 列舉各種垢而以無明為極垢；（4）dhp245-250 處理無恥 vs 有恥、酒、財施之定；（5）dhp251-255 述漏盡者之定、己過難見、賭徒藏骰之喻。

## 段落註釋

### dhp235-238：黃葉、閻摩使者、作燈

「閻摩使者」（Yama-purisa）為死神閻摩之使者，佛教文獻中常以之喻老死之逼近。「資糧」（sambhāra）為「道之資糧」（magga-sambhāra），含戒定慧三學。「作燈」（dīpamattano, 自作之燈）為《法句》著名比喻：自依止（attā upātika）、法依止（dhamma upātika），而非他依止。學界（Bodhi 2005, pp. 121-123）指出此喻亦見於 *Mahāparinibbāna Sutta*（DN 16）佛陀最後遺教之「自作洲 / 自作燈」。「閻摩之境」（Yama-rājja）與「聖者之地」（ariya-bhūmi）對照，cf.《增支部》之「聖 / 凡」二階分判。

### dhp239：銀匠除垢之喻

「漸次、時時一點一點，如銀匠除銀之垢，當除己之垢」— 學界（Carter & Palihawadana 1987, p. 196）解為「漸修」之經典比喻，cf. SN 22.101 之「垢隨眠可漸斷」之說。此喻亦見於中阿含、巴利《增支部》之相應經。

### dhp240-241：鐵鏽自蝕之喻

「如鐵生起之鏽，自鏽起而蝕鐵；如是多作惡行者，自身之業引向惡趣」— 此「業自招」之喻為佛教業論之通俗表述。「惡趣」（duggati）含地獄、餓鬼、畜生三惡道。

### dhp242-244：垢之列舉

學界將此三頌分為「垢之七例」與「極垢之無明」：dhp242 列「不誦咒 / 不勤家宅 / 懈怠 / 放逸」四垢（多解為在家者之垢）；dhp243 列「婦女之惡行 / 施者之慳吝 / 惡法」三垢；dhp244 以「比諸垢更垢者，無明是極垢」總結。無明（avijjā）為四聖諦中「苦、集、滅、道」之首（「苦諦」之根），《法句》此處以「極垢」定位之。Bhikkhu Bodhi（2005, p. 124）指出此處「無明」之強調反映《法句》與《相應部》之共見，即無明為十二緣起之首。

### dhp245-250：無恥 vs 有恥、酒

「無恥者易活，如烏鴉啄食；躁動者、傲者，以污穢而活」vs「有恥者難活，常求清淨者 ... 清淨而活之見者」— 此對比展現佛教「重淨」之倫理觀。「酒」（surā）與「迷醉之飲」（meraya-pāna）為古代印度之兩類酒精飲料，*Dhp-a* 註：surā 為發酵穀酒、meraya 為花果釀酒。此兩者被列為五戒之第四戒（不飲酒）之主要對象。

dhp250「人隨信而施 ... 彼於晝夜不得定」— 學界（Pathak 2017）解為「施者若於他人食飲生不悅意（疑嫉、悔施）」則心不得定。cf. AN 6.37 之「不悔施則心得定」之相應說。

### dhp251-253：火、牢、網、河四喻

「無有貪等之火、無有瞋等之牢、無有痴等之網、無有渴愛等之河」— 此四喻為《法句》著名的「煩惱喻」系列：貪如火（燃燒）、瞋如牢（繫縛）、痴如網（纏裹）、渴愛如河（漂溺）。*Dhp-a* 註：此四者為三不善根（貪瞋痴）與集諦（渴愛）之合併喻示。學界（Appleton 2014）認為此四喻反映了佛教對「煩惱動力學」之形象化思考。

### dhp254-255：己過難見、賭徒藏骰之喻

「他人之過易見、己之過難見 ... 如賭徒藏骰（kali）」— 「藏骰」喻為《法句》特有之通俗比喻。*Dhp-a* 註：「藏骰」指賭徒於贏時亮出、於輸時藏匿。學界（Carter & Palihawadana 1987, p. 215）認為此喻精準地描繪人類「自利偏差」（self-serving bias）：對他人之過放大、對己之過隱藏。

## 名相索引

- **mala**（巴；梵 mala）— 垢、穢、染污；本章核心詞
- **avijjā**（巴；梵 avidya）— 無明；十二緣起之首
- **Yama-purisa**（巴）— 閻摩使者；死神之使者
- **duggati**（巴；梵 durgati）— 惡趣；三惡道之總稱
- **surā / meraya-pāna**（巴）— 穀酒 / 花果釀酒；佛教「不飲酒」戒之主要對象
- **taṇhā**（巴；梵 tṛṣṇā）— 渴愛；四聖諦中集諦之核心
- **duccarita**（巴）— 惡行；身語意各二 / 三 / 三（合為十惡）
- **kali**（巴）— 骰子；此處喻「藏匿」

## 與其他經典的關聯

- dhp235-236 之「自作燈」與《長部 16 經》（DN 16, *Mahāparinibbāna Sutta*）之最後遺教直接相應。
- dhp239 之「銀匠除垢」與《相應部 22.101 經》（SN 22.101）之漸修說相應。
- dhp244「無明是極垢」與《相應部》之「無明為先導」說相應（SN 12.1 等）。

## 主要學術爭議

1. **「無恥 / 有恥」（ahirika / anottappa）之概念史**：學界爭議其是否屬「煩惱」（kilesa）之列。*Visuddhimagga*（XXII）將其列為「煩惱之伴」（kilesa-paṭibhāga）；學界（Gethin 2001）接受此分判，但指出其並非「根」而是「隨眠」。
2. **dhp253 之四喻順序**：為何「貪 → 火」、「瞋 → 牢」、「痴 → 網」、「渴愛 → 河」？*Dhp-a* 未明示理由；學界推測與各煩惱之「作用模式」相關（燃、繫、纏、流）。

=== 19 | 法住品（Dhammaṭṭhavagga, dhp256-272）— 註釋 ===

## 章旨

本章以「誰是真持法者」為核心議題，連續辨析「持法者（dhammaṭṭha）/ 智者（pandita）/ 長老（thera）/ 善人（sādhurūpa）/ 沙門（samaṇa）/ 比丘（bhikkhu）/ 牟尼（muni）/ 聖者（ariya）」八個身分稱號之真實義涵，皆以「非以 X，為 Y；唯以 Y，為 Y」之否定公式展開，為《法句》中倫理辨析最系統之篇章。

## 段落註釋

### dhp256-257：持法者之判

「以無暴之法（asāhasena dhammena）、以平等引導他者、護法之慧者，稱為持法者」— 「無暴之法」學界（Norman 1997, p. 53）解為「非以暴力 / 非以強迫」；與佛教之「依法」（dhamma-vinaya）治僧原則一致。「持法者」（dhammaṭṭha）為古代印度常用之政治與宗教術語；佛教採用之而賦予「以法為依」之新意涵。

### dhp258-259：智者非以多語、持法者非以多語

兩頌連續以「非以多語」辨析「智者」與「持法者」。學界指出此反映了早期佛教社群中「重質不重量」之口語倫理觀；cf. *Itivuttaka* 100 之「口四德」（說實、說時、說柔軟、說有益）。

### dhp260-263：長老 vs 善人

dhp260「非以白髮為長老；彼雖年老，但名徒老（mogha jiṇṇa）」— 「徒老」（空老）為《法句》特有之貶義詞，與 *Theragāthā* 中對「無實修之長老」之批評相應。「長老」（thera）原意「年長」，佛教重新定義之為「具戒律、慧、定、解脫」之具足者。dhp262-263「善人非以語巧色相」亦同樣以「外在」vs「內在」之對比。

### dhp264-267：沙門 vs 比丘

dhp264「非以剃髮、語虛妄，為沙門；具欲貪者，豈是沙門」— 「沙門」（samaṇa）原為印度「出家修行者」之通稱；佛教重新定義之為「能止惡者」。此辨析與佛教「四沙門果」（sotāpanna / sakadāgāmi / anāgāmi / arahant）之分類呼應。dhp266-267「非以乞食他人為比丘」對比「捨善與惡、修梵行」之真比丘；cf. *Theragāthā* 對「乞食比丘」之自我反思。

### dhp268-269：牟尼非以寂默

「非以寂默（mona）、鈍根無智為牟尼；如秤取捨（tulamva paggayha），取勝而從（varamādāya）者為智者」— 「如秤取捨」為《法句》著名比喻。學界（Carter & Palihawadana 1987, p. 230）認為此喻反映了佛教對「中道」（majjhimā paṭipadā）之形式化思考：非此非彼、非偏非倚，唯依「慧」（paññā）取捨。

### dhp270-271：聖者非以戒禁

「非以戒禁（sīlabbata）、或多聞（bāhusacca）；亦非以得定（samādhi）、或獨居（vivitta-sayana）」— 此頌明確反對「戒禁取」（sīlabbata-parāmāsa），為佛教對「外道」之常見批評之一。「戒禁取」指執著於形式戒律為究竟解脫之見，cf. *Brahmajāla Sutta*（DN 1）之六十二見批評。

### dhp272：比丘之漏盡之信

「我得出離之樂（nekkhamma-sukha），非凡夫所行；比丘得信（vissāsa），未得漏盡」— 此頌呈現「有信而未得漏盡」之中間狀態。學界（Cousins 1996）指出此反映「信（saddhā）與慧（paññā）」之配合：信為凡夫入流之始，但漏盡需慧之圓滿。

=== 20 | 道品（Maggavagga, dhp273-289）— 註釋 ===

## 章旨

本章為《法句》中理論性最強之篇章，系統闡述「八支正道」、「四聖諦」、「三法印（無常、苦、無我）」、「有 / 無有」（bhava / vibhava）二邊中道等核心教義。結構為：（1）dhp273 開篇以「八支、四足、離欲、有眼者」之四最勝；（2）dhp274-276 述「唯一道」、「拔刺」；（3）dhp277-279 述三法印；（4）dhp280-282 述精勤與三業道；（5）dhp283-285 述斷林與秋蓮之喻；（6）dhp286-289 述障難、淨往涅槃。

## 段落註釋

### dhp273：四最勝

「八支（aṭṭhaṅgika）最勝」即八正道（正見、正思惟、正語、正業、正命、正精進、正念、正定）。「四足（caturo padā）」為四聖諦之別名。「離欲」（virāga）為涅槃之異名之一（cf. dhp187, dhp225）。「有眼者」（cakkhumā）為佛陀之異名，亦為「具慧者」之通稱。*Dhp-a* 解此頌為「於諸道中，八支正道最勝；於諸諦中，四聖諦最勝；於諸法中，離欲最勝；於二足（有足）中，有眼者（佛）最勝」。

### dhp274-276：唯一道、拔刺

dhp274「此是唯一道（natthañño），無有他；為見清淨（dassana-visuddhi）；汝等當行此，此是魔之迷惑（pamohana）」— 「唯一道」之強調反映了早期佛教對「八正道」之絕對肯定；「魔之迷惑」指將「他道」（非八正道）視為解脫道之誤認。「拔刺」（sallakantana）喻見 dhp285, dhp308（地獄）。

dhp276「行道者得解脫、禪修者（jhāyin）脫魔縛（māra-bandhana）」— 「禪修者」一詞之使用反映了《法句》中「瑜伽」（yoga, dhp280-282）與「禪那」（jhāna）之並行。

### dhp277-279：三法印

此三頌連續陳述「三法印」（tilakkhaṇa）：「一切行無常（anicca）」「一切行是苦（dukkha）」「一切法無我（anattā）」。「厭離於苦、此是清淨道」之結句被視為「三法印 + 清淨道」之標準表述。學界（Thérèse-Anne Druart, in *Buddhist Philosophy*, 2017）指出：「三法印」之概念在巴利《尼柯耶》中已具雛形，但定型於部派佛教之「法印」（dhamma-uddesa）概念。

### dhp280：當起時不起

「當起時不起、年少壯健而懈怠；意頹廢、懶惰，懈怠者以慧不見道」— 此「年少壯健」之詞呼應 dhp235 之「黃葉、閻摩使者」；強調「精勤」之緊迫性。cf. *Mahāparinibbāna Sutta*（DN 16）之「年少壯健、有精力、有氣力 ... 不應放逸」。

### dhp281：護語、三業道

「護語、以意善自制、不以身行惡；當淨此三業道、當證聖者所說之道」— 此頌呼應 dhp234 之「身語意三業防護」，並明指「淨三業道」為「證聖道」之實踐。「kammapatha」（業道）一詞專指「身三、語四、意三」之十業道（含善與不善兩端）。

### dhp282：瑜伽 / 無瑜伽

「瑜伽（yoga）生廣大、無瑜伽（ayoga）是廣大之滅；知此雙徑（dvedhāpatha），為有（bhava）與無有（vibhava）；當如是安立自身，使成廣大增長」— 此頌為《法句》中最具理論深度者之一。學界（Bronkhorst 1993）認為此處「瑜伽 / 無瑜伽」之「雙徑」即「有（流轉） / 無有（還滅）」之二邊。佛教主張「中道」離此二邊：「有」為常見，「無有」為斷見。cf. MN 26 之「離有 / 無有」中道說。

### dhp283-285：斷林、秋蓮

dhp283「當斷林（vana）莫斷樹（rukkhā），從林生怖畏」— 學界（Norman 1997）解「林」為「欲林」（vanatha）之略，即「煩惱叢林」之喻。「斷林不斷樹」喻「斷煩惱而不斷生命」。dhp284「如小牛、飲乳之犢於母」— 此為「欲繫」之俗喻，與《中阿含》之相應喻示同。dhp285「當斷己之貪愛，如秋蓮（sāradika kumuda）以手」— 「秋蓮以手」喻「易於拔除」；cf. *Udāna* 之相應喻示。

### dhp286-289：愚者不知障難、淨往涅槃

dhp286-287「我於此雨季住 ... 愚者如是思惟，不覺障難」— 「障難」（antarāya）為五種死之障難：蛇、蠍、人非人、跌墮、飲食不當。*Dhp-a* 解為「愚者只見外在住處之險，未覺生命本身之障難」。

dhp289「知此義已、具戒慧者，當速淨往涅槃之道」— 此結句與 dhp21「不放逸者不死、懈怠者如死」之精神一貫，為本章亦為《法句》整體之核心教義。

## 主要學術爭議

1. **「三法印」是否為佛陀原始教說**：學界（Bronkhorst 1993; Druart 2017）爭議：（a）三法印之定型化屬部派佛教時期；（b）但其概念內涵於原始《尼柯耶》中已具雛形。《法句》此處之表述屬中間形式。
2. **dhp282「有 / 無有」之詮釋**：學界爭議「vibhava」是否等同「斷見」（uccheda-diṭṭhi）。*Dhp-a* 將「vibhava」解為「欲無」（kāma-vibhava）；學界（如 Jayatilleke 1963）認為此處之「有 / 無有」更接近形上學之「存在 / 非存在」對立。
3. **dhp285「秋蓮」之地域考證**：*sāradika*（秋季）與 *kumuda*（白蓮）為印度恆河流域之秋季物產；學界認為此喻反映了《法句》之地區性特色。

=== 21 | 雜品（Pakiṇṇakavagga, dhp290-305）— 註釋 ===

## 章旨

本章以「雜」為名（Pakiṇṇaka = 雜多），內容涵蓋少樂 / 大樂之取捨、「殺母殺父」之著名隱喻、六頌善覺瞿曇弟子、難出家難住、最後以「一座一臥獨行」作結。

## 段落註釋

### dhp290：捨少樂取廣大樂

「若見捨少樂（mattāsukha），有廣大之樂（vipula sukha）；智者當捨少樂、見後取廣大樂」— 「少樂」vs「廣大樂」之對照反映了佛教對「現法樂 vs 究竟樂」之衡量。學界（Carter & Palihawadana 1987, p. 243）認為此頌為「出離」（nekkhamma）教義之通俗表達。

### dhp291：逼惱他人者自樂

「以逼惱他人之苦（paradukkhūpadhāna），欲自樂者；與怨憎會（verasaṁsagga）者，彼不離於怨」— 「怨憎會」為八苦之一；《法句》此處以「與怨憎會」喻「逼惱他人者反自陷怨」。

### dhp292-293：應作不應作、身念處

dhp292「所應作捨、不應作反作；驕慢、放逸者，彼等漏增長」— 「應作」（kicca）vs「不應作」（akicca）之辨為《法句》常見主題（cf. dhp116, dhp312）。dhp293「若彼善勤修、常身念處（kāyagatā sati）；彼不行不應作、於所作恆常精勤；於覺者、正知者（sampajāna），漏得以滅盡」— 「身念處」為四念處（身、受、心、法）之首；「正知」（sampajāna）為《法句》重要概念，指「清楚認知」。

### dhp294-295：殺母、殺父、二王種、婆羅門

此二頌為《法句》最具爭議之隱喻，學界通稱為「殺父母喻」（Parents-killing verses）。*Dhp-a* 註解（Buddhaghosa 解為）：

- **母** = 渴愛（taṇhā）
- **父** = 邪見（micchā-diṭṭhi）
- **二王種** = 斷見（uccheda-diṭṭhi）與常見（sassata-diṭṭhi）
- **婆羅門**（第五）= 執著於「我是」（asmi-māna，我慢之極）

學界（Bodhi 2005, pp. 152-153; Norman 1997）指出此隱喻為早期佛教「煩惱人格化」之典型思維。「殺」並非字面之殺，而是「斷除」（pahāna）。*Dhp-a* 引《彌蘭王問經》（Milindapañha）之相應解釋印證之。

### dhp296-301：六頌「善覺瞿曇弟子」

此六頌以「善覺（suppabbuddha）之覺醒、常瞿曇（Gotama）弟子」之開頭連續六次，每次結尾不同：「常佛所念（buddhagatā sati）/ 常法所念 / 常僧所念 / 常身念處 / 意樂於不害（ahiṁsāya rata）/ 意樂於修習（bhāvanāya rata）」。學界指出此「六變」為《法句》之修辭手法；cf.《增支部》之「隨念佛、法、僧、戒、施、天」六隨念。六頌結尾呈現「三寶念 + 三學（戒定慧）」之六門。

### dhp302-304：難出家、難住

dhp302-304 連續三次「難 ... 」之排比：「難出家、難住樂、家宅難住是苦；與不善者共住苦、從苦出離者難得」— 反映早期佛教對「在家 vs 出家」之兩難思維。「故莫為出離者、莫隨苦行」（dhp304）為「中道」之另一表述：「出離」非「苦行之極」。

### dhp305：一座一臥、獨行

「一座、一臥、獨行不懶；獨自調御己身，於林中當自娛」— 此頌呼應 *Theragāthā* 中諸多「獨行林中」之自敘。學界（Cousins 1996）認為此反映了「阿蘭若行」（āraññavāsa, 林居）為早期佛教重要之修行傳統。

- dhp305「一座一臥獨行」與《長老偈》（*Theragāthā*）之林居傳統相應。

## 主要學術爭議

1. **「殺父母」喻之詮釋史**：*Dhp-a* 之「煩惱人格化」解為最權威之 Theravāda 傳統；學界（如 Jayatilleke 1963）認為此喻可能亦承自印度古傳之「道德修辭」（倫理化比喻），非佛教獨創。
2. **dhp304 之「莫為出離者」是否反出離**：學界爭議此是否為「中道」之強調，抑或為對「過度出家」之批評。*Dhp-a* 解為後者。

=== 22 | 地獄品（Nirayavagga, dhp306-319）— 註釋 ===

## 章旨

本章以「地獄」為核心議題，兼及邪見與正見之對比。結構為：（1）dhp306-309 連續列舉導致地獄之惡行；（2）dhp310-313 述自護與懈怠之果報；（3）dhp314「邊境城」之自護喻；（4）dhp315-319 述「邪見四句」與「正見」。

## 段落註釋

### dhp306-308：說非真者、披黃褐領者、食熱鐵丸

dhp306「說非真者赴地獄，作已言我未作」— 「說非真者」（musāvādī, 妄語者）為十惡中語四之首。「二人死後相等」之表述學界（Carter & Palihawadana 1987, p. 253）解為「妄語者與作已言未作者同受地獄報」。

dhp307「披黃褐領者眾多，惡法者無自制」— 「黃褐」（kāsāva）為佛教僧衣之色；此頌批判「著僧衣而不行僧事」之流。cf. *Mahāparinibbāna Sutta*（DN 16）對「末世比丘」之類似批評。

dhp308「食熱鐵丸為勝，如火燄相似；破戒者若食，國施食無自制」— 學界指出此反映了佛教「在家 vs 出家」之倫理對比：在家者受國施食而行惡，較出家者食熱鐵丸更為不當。

### dhp309-310：放逸之墮

dhp309「放逸之人四處墮，親近他人之婦；不得善利不得眠，第三毀謗第四地獄」— 「親近他人之婦」（paradāra）為十惡中身三之一（邪淫）。「四處墮」學界解為「善利、眠、毀謗、地獄」四項損失。

### dhp311：茅草之喻

「如茅草執持不善，割傷其手；沙門律儀執持不善，將牽至地獄」— 「茅草」（tina）喻「沙門律儀」（samaṇadhamma）。*Dhp-a* 解為：執持不善之沙門律儀反招地獄報。學界（Bodhi 2005, p. 159）認為此喻反映了佛教對「形式宗教 vs 實質修行」之區分。

### dhp312-313：懈怠之業

dhp312「凡懈怠之業，染污之戒；雜亂之梵行，不得大果」— 「染污之戒」（duṭṭhulla-sīla）為《增支部》（AN 2.7 等）所批評之「染污戒」。dhp313「作已當作之，堅固而努力；懈怠之遊行者，徒增塵埃」— 「徒增塵埃」之表述反映了對「形式宗教生活」之批評。

### dhp314：邊境城喻

「猶如邊境城，內外皆防護；如是當自護，剎那勿錯過；失剎那者悲泣，置於地獄」— 此為《法句》著名比喻：「剎那」（khaṇa）為最短時間單位。「失剎那者悲泣，置於地獄」呼應 dhp235-236 之「閻摩使者」催促語氣。

### dhp315-319：邪見四句 vs 正見

此五頌以「於 X 見非 X / 於非 X 見 X / 執邪見 / 執正見」之四句結構展開邪見之四相與正見之正相：

- dhp315：「於無恥者恥，於有恥者不恥；執邪見者，眾生墮惡趣」
- dhp316：「於無畏見畏，於畏見無畏；執邪見者，眾生墮惡趣」
- dhp317：「於無過見過，於過見無過；執邪見者，眾生墮惡趣」
- dhp318：（同 315）
- dhp319：「知過為過，知非過為非過；執正見者，眾生生善趣」

學界指出此四句之結構反映了佛教「見」（diṭṭhi）之分類：「顛倒見」（vipariyāsa-diṭṭhi）vs「如實知見」（yathā-bhūta-ñāṇa-dassana）。dhp319 為結語，呼應 *Kālāma Sutta*（AN 3.65）之「不依傳統、不依權威、唯依現量與法」原則。

## 名相索引

- **musāvāda**（巴；梵 mṛṣāvāda）— 妄語；十惡中語四之首
- **kāsāva**（巴；梵 kāṣāya）— 黃褐；佛教僧衣之色
- **paradāra**（巴；梵 paradāra）— 他人之婦；邪淫之對象
- **samaṇadhamma**（巴）— 沙門律儀；沙門之修行法
- **duṭṭhulla-sīla**（巴）— 染污之戒；與「清淨戒」（pārisuddhi-sīla）對
- **khaṇa**（巴；梵 kṣaṇa）— 剎那；最短時間單位
- **vipariyāsa-diṭṭhi**（巴）— 顛倒見；四顛倒（常、樂、我、淨）之執
- **diṭṭhi**（巴；梵 dṛṣṭi）— 見、見解
- **duggati**（巴；梵 durgati）— 惡趣

## 與其他經典的關聯

- dhp307「披黃褐領者」之批判與《長部 16 經》（DN 16）之末世比丘批評同調。
- dhp314「邊境城」喻與《相應部 35.204 經》（SN 35.204, *Āditta Sutta*）之「邊境城」喻相應。
- dhp319「正見」之強調呼應《中部 9 經》（MN 9, * Sammādiṭṭhi Sutta*）之專論。

## 主要學術爭議

1. **「地獄」（niraya）之概念史**：學界爭議佛教之「地獄」概念是否承自印度古傳（吠陀後期之 *garbhādhāna* 儀軌已提及 *naraka*）。*Dhp-a* 詳細描述地獄結構；學界（如 Obeyesekere 2003）認為此反映了佛教在地獄描述上之印度文化適應。
2. **「失剎那」（khaṇa-vidhāna）之時長**：*Dhp-a* 解為一呼吸之間；學界（Appleton 2014）指出此與《增支部》（AN 6.29）之「六剎那」說呼應。

=== 23 | 象品（Nāgavagga, dhp320-333）— 註釋 ===

## 章旨

本章以「象」為核心喻象，喻示修行人之調御、獨行與堅忍。結構為：（1）dhp320-324 述「戰場之象」、「調御者」、「自調御更勝」；（2）dhp325-327 述「堅牢之象 / 大豬」、「鈎師制象」、「拔出泥中」；（3）dhp328-330 述「賢善之友 vs 不得」、「獨行最勝」；（4）dhp331-333 述「所得 / 命終 / 敬父母 / 持戒」四樂。

## 段落註釋

### dhp320-323：戰場之象、信度良種馬

dhp320「我如戰場之象，箭從弦落；我忍辱於惡語，破戒者實多」— 學界（Carter & Palihawadana 1987, p. 264）指出此頌為《法句》中少見之「自敘語氣」（「我如 ... 」），反映了佛教對「忍辱」（khanti）之強調。「破戒者實多」之表述為對末世比丘之批評。

dhp322「馬中調御者為勝，信度良種馬（ Sindhava, 信度）」（信度即 Sindhu，今巴基斯坦信德省 Sindh）；古代印度之 Sindhava 馬為最優駿馬之一。

### dhp323-324：自調御更勝、堅牢之象

dhp323「非此等乘具，能達未曾至方；如自善調御者，調御者乘調御者」— 「未曾至方」為涅槃之異說。學界（Appleton 2014）解此為「自調御者能達涅槃，他物乘具不能達」之對比。dhp324「名堅牢之象，破柵難制；繫住不食，象憶念象林」— 此頌反映了古印度「馴象」之傳統；cf.《本生》之相關描述。

### dhp325-327：大豬、鈎師、泥中之象

dhp325「當懈怠而大食，沈睡翻轉；如飼於坑之大豬，愚者再三入胎」— 「再三入胎」喻「輪迴」。「坑之大豬」喻「食、睡、欲」之愚者。dhp326「昔此心遊行，隨意隨欲隨樂；今日我當善調伏，如鈎師制破象」— 「鈎師」（aṅkuśika）為古印度之象師，以「鈎」（aṅkuśa）制象。學界指出此「鈎」喻「正念」（sati）與「正知」（sampajāna）。

dhp327「當喜不放逸，守護自心；如陷泥中之象，拔出己於難渡」— 「難渡」（duggati）為「惡趣」之同音異字；此處亦含「險難」之意。

### dhp328-330：賢善之友、獨行

dhp328「若得賢善之友（kalyāṇa-mitta），同行善行堅固；越諸險難，應與彼同喜正念而行」— 「賢善之友」（kalyāṇa-mitta / sappurisa）為佛教重要概念，《增支部》（AN 8.54）詳論「賢善之友」之八德。「同喜」（sammodamāna）為「共住」之意。dhp329「若不得賢善之友 ... 如王捨棄征服國，獨行如象入林」— 此頌採「國王征服國」之喻。dhp330「獨行最勝，愚者無友情；獨行不作惡業，寂靜如象入林」— 「獨行」（ekavihāra）為阿蘭若行之核心；cf. *Theragāthā* 中諸多獨行詩。

### dhp331-333：所得 / 命終 / 敬父母 / 持戒四樂

dhp331「有所得時友為樂，隨緣喜為樂；命終時福為樂，捨一切苦為樂」— 此四樂之表述反映佛教「以慧衡量」之倫理觀。dhp332「世間敬母為樂，敬父亦為樂；世間沙門律儀為樂，梵行亦為樂」— 此處「敬父母」之「樂」與一般世俗理解略異，*Dhp-a* 解為「修行者能奉事父母」之福報。dhp333「持戒至老為樂，信確立為樂；得智慧為樂，不作惡為樂」— 此為《法句》著名之四樂，呼應 dhp15「善護心 ... 得不死」之教義。

## 名相索引

- **Sindhava**（巴；梵 Saindhava）— 信度良種馬；產自 Sindh（今巴基斯坦）
- **aṅkuśika**（巴；梵 aṅkuśika）— 鈎師；以鈎制象者
- **kalyāṇa-mitta / sappurisa**（巴）— 賢善之友
- **sammodamāna**（巴）— 同喜、共住
- **ekavihāra**（巴）— 獨行；阿蘭若行
- **khanti**（巴；梵 kṣānti）— 忍辱
- **sati / sampajāna**（巴）— 正念 / 正知

## 與其他經典的關聯

- dhp320「戰場之象」之喻與《相應部》（SN）之「象喻」相應；cf. SN 11.3 等。
- dhp328-330「賢善之友」之強調與《增支部 8.54 經》（AN 8.54, *Kalyāṇamitta Sutta*）之專論相應。
- dhp331-333 四樂之排列呼應 *Theragāthā* 中諸多「喜樂」（pāmojja）之教說。

## 主要學術爭議

1. **「象」之文化象徵**：學界指出古印度對「象」之雙重態度：象為「王者坐騎」與「林中兇獸」。《法句》此處借用「戰場之象」喻「忍辱之修行人」；cf. *Theragāthā* 中諸多「象詩」（如 Nāgasamāla, Nāgita）。
2. **dhp332「敬父母」之文化背景**：古印度傳統重孝道（cf. *Taittirīya Upaniṣad* 1.11）；《法句》此處將「敬父母」列入「樂」之列，反映佛教對傳統孝道之兼容而非否定。

=== 24 | 愛欲品（Tanha vagga, dhp334-359）— 註釋 ===

## 章旨

本章為《法句》中對「愛欲 / 渴愛」之系統處理，篇幅最長（dhp334-359, 26 頌）。結構為：（1）dhp334-338 述「蔓增長 / 荳蔻 / 蓮葉水珠 / 掘根」之喻；（2）dhp339-341 述「根未斷 / 三十六流」；（3）dhp342-344 述「憶念親愛 / 渴所驅」；（4）dhp345-349 述「愛森林 / 鐵木草繩 / 捕猿網 / 前捨後捨」；（5）dhp350-353 述「思惟壓伏 / 不淨觀 / 達目的無畏」；（6）dhp354 述「法施勝一切施」；（7）dhp355 述「財物殺愚者」；（8）dhp356-359 述「田有草害」之四種施之果報。

## 段落註釋

### dhp334-336：蔓、荳蔻、蓮葉水珠

dhp334「人之放逸行者，愛欲如蔓增長」— 「蔓」（latā）為古印度常見植物喻，喻「愛欲之擴張性」。dhp335「其悲增長，如盛長之荳蔻」— 「荳蔻」（*vanatthu*, 巴；對應 *syonāka* 或 *kāravellikā*）為印度常見之灌木。*Dhp-a* 註：「荳蔻」喻「愛欲之叢林」。

dhp336「如蓮葉水珠」— 此為《法句》著名之清淨喻：水珠不染蓮葉。cf. *Udāna* 中「蓮葉水珠」之相應比喻；學界（Carter & Palihawadana 1987, p. 273）指出此喻亦見於巴利《本生》之相應場景。

### dhp337-338：掘荳蔻根、溪流蘆葦

dhp337「當掘愛欲之根，如掘荳蔻根；莫如溪流之蘆葦，為魔一再斷」— 此頌呼應 dhp283-285 之「斷林」喻。「一再斷」喻「魔」（Māra）反覆斷之。

### dhp339-341：根未斷、三十六流

dhp339「如根未斷樹堅固，斷已還生；如是愛欲隨眠未斷，此苦一再生」— 「隨眠」（anusaya）為佛教煩惱論之重要概念，指「潛伏之煩惱」。學界（Cousins 1996; Gethin 2001）指出「隨眠」之七種（七隨眠）為部派佛教定型。

dhp340「三十六流（sota），多生悅意；依貪之思，大流注邪見者」— 「三十六流」之具體所指學界爭議：*Dhp-a* 解為「三十六種欲流」；現代學者（如 Jayatilleke 1963）認為可能指「三十六種愛欲之對象」或「愛欲之分類」（cf. AN 6.63 之六十二見中含類似的分類傳統）。

### dhp342-344：憶念親愛、渴所驅（繫兔）

dhp342「憶念與親愛，人之喜樂；彼享樂求樂者，乃為生老所困之人」— 「憶念」（sati 廣義）與「親愛」（piya / pema）為上一章（16）主題之延續。dhp343-344「渴所驅之眾生，如繫兔四處遊；結縛所纏，長時間一再受苦」— 「繫兔」（migo')喻「渴愛之驅動」。*Dhp-a* 解：渴愛如繫兔，雖斷其繩仍四處逃。學界指出此喻反映了古印度「狩獵」之常見經驗。

### dhp345-349：愛森林、鐵木草繩、捕猿網

dhp345「彼愛森林耽森林，出森林仍奔森林；應見此人，出繫縛仍奔繫縛」— 「森林」（vana）喻「愛欲之叢林」。

dhp346-347「智者不言此為堅縛，鐵木草繩；著於珠寶耳飾，於妻子等之貪著。智者說此為堅縛，牽下鬆而難解；斷此亦遊行，無貪而捨欲樂」— 此二頌為《法句》著名比喻：「鐵木草繩」喻「外在束縛」（易斷），「珠寶耳飾」喻「內在貪著」（難斷）。*Dhp-a* 註：「牽下鬆而難解」指「貪著欲樂之人心雖暫鬆仍難解脫」。學界（Norman 1997, pp. 60-61）指出此喻反映了佛教對「心縛」之重視遠超「身縛」。

dhp348「貪染而隨流者，如自製之捕猿網」— 「捕猿網」為古印度「獵猴」之常用工具；cf.《本生》之相應描述。

dhp349「前捨後捨，中亦捨，越存在者；處處心解脫，不再受生老」— 此頌呼應 dhp282 之「瑜伽 / 無瑜伽」。「越存在者」（bhava-pañña）學界解為「已越『有』者」，即證得涅槃者。

### dhp350-353：思惟壓伏、不淨觀

dhp350「思惟壓伏之人，強貪見淨；愛欲更增，此實作堅縛」— 「思惟」（vitakka）指「未鎮伏之心念」。「強貪」（pañcaka）指「五種感官貪著」。dhp351「樂於止息思惟，常正念修不淨（asubha）；此實終焚，此斷魔縛」— 「不淨觀」（asubha-bhāvanā）為四念處中「身念處」之主要修法；cf.《中部》（MN）之相應教說。「此實終焚」喻「此（不淨觀）燒盡愛欲」。

dhp352-353 述「達目的無畏」、「離欲無取」、「此最後身」、「自證知更何指示」— 「自證知」為佛教「自知自證」（sayam abhiññā sacchikatvā）之標準表述；cf.《中部》（MN 4, 36 等）之「自知自證」。

### dhp354：法施勝一切施

「法施勝一切施，法味勝一切味；法樂勝一切樂，愛盡勝一切苦」— 此頌為《法句》中對「法施」（dhamma-dāna）之最高推崇。學界（Cousins 1996）指出「法施」之強調反映了佛教「法為主」（dhamma-eena）之核心理念。「財施」（āmisa-dāna）vs「法施」（dhamma-dāna）之對比亦見於《增支部》（AN 2.18）。

### dhp355：財物殺愚者

「財物殺愚者，不求彼岸者；愚者因財貪，自殺其身」— 「愚者因財貪，自殺其身」之說反映了佛教對「貪財」之嚴厲批評。*Dhp-a* 解：愚者因貪財而破戒、入地獄。

### dhp356-359：田有草害（四種施之果報）

此四頌結構相同：「田有草害，此眾生有 X 害；是故施於離 X 者，得大果」（X = 貪 / 瞋 / 癡 / 欲）。*Dhp-a* 解：「田」喻「眾生」；「草害」喻「煩惱」；「離 X 者」喻「已斷煩惱之聖者」。施予已離煩惱之聖者，其福報為最大。

此結構呼應 dhp108-115（亦以「田有草害」之喻論施）。學界（Appleton 2014）認為此「田喻」為《法句》最完整之「四施」分類。

## 名相索引

- **taṇhā**（巴；梵 tṛṣṇā）— 渴愛；本章核心詞；亦作「欲 / 貪」
- **anusaya**（巴；梵 anuśaya）— 隨眠；潛伏之煩惱
- **sota**（巴）— 流；此處指「欲流」
- **vana / latā**（巴）— 森林 / 蔓；愛欲之喻
- **asubha**（巴；梵 aśubha）— 不淨；不淨觀之對象
- **vitakka**（巴；梵 vitarka）— 思惟；未鎮伏之心念
- **dhamma-dāna / āmisa-dāna**（巴）— 法施 / 財施
- **sayam abhiññā sacchikatvā**（巴）— 自知自證
- **kalyāṇa-mitta**（巴）— 賢善之友（與第 23 章呼應）
- **bhava-pañña**（巴）— 越有者；已證涅槃者

## 與其他經典的關聯

- dhp339「隨眠未斷還生」與《相應部》（SN）之「七隨眠」說相應；cf. AN 7.11 之「七隨眠」。
- dhp351「不淨觀」與《中部》（MN 119, *Kāyagatāsati Sutta*）之相應教說同源。
- dhp354「法施勝一切施」呼應《增支部 2.18 經》（AN 2.18, *Āmisa Sutta*）之「法施 vs 財施」比較。
- dhp356-359「田有草害」與 dhp108-115 之相應比喻構成《法句》之「施之分類」完整教說。

## 主要學術爭議

1. **「三十六流」之具體所指**：*Dhp-a* 解為「三十六種欲流」；學界爭議其可能：（a）為象徵性數字（cf. AN 之類似的象徵數字使用）；（b）為佛教對印度古傳「三十六」之沿用。Jayatilleke（1963）認為後者可能性較高。
2. **「隨眠」（anusaya）之概念史**：學界（Gethin 2001）指出「七隨眠」（貪、瞋、痴、慢、見、疑、戒禁取）為部派佛教定型，但《法句》此處之使用為較早期形式，尚未明確列舉七種。
3. **dhp354「法施」之優先性**：此說是否反對「財施」？學界（Bodhi 2005, p. 176）解為「法施之效益大於財施」，非「反對財施」；cf. AN 2.18 之「法施為最勝，但財施亦隨喜」之精神。
4. **dhp356-359 之四施結構**：與 dhp108-115 之同喻重出，反映《法句》編纂者有意識之分類整理；學界（Carter & Palihawadana 1987）認為此反映「口傳傳統中常見之詩文重組」。

## 本段（章 16-24）主要學術爭議彙整

### 1. 「五重執著」（piya / pema / rati / kāma / taṇhā）

學界主流（Norman 1997; Bodhi 2005）認為此五詞非隨意反覆，而是層次分明的五重分類：piya（可愛）→ pema（摯愛）→ rati（喜）→ kāma（欲）→ taṇhā（渴愛），愈往後愈抽象、愈廣義。少數學者（如 Pathak 2017）認為部分詞為同義反覆以強化記憶，未必有意識分類。

### 2. 「殺父母」喻（dhp294-295）之詮釋史

*Dhp-a* 之「煩惱人格化」解（母 = 渴愛、父 = 邪見、二王種 = 斷常二見、婆羅門 = 我慢）為 Theravāda 傳統定本。學界爭議：

- 是否承自印度古傳之「道德修辭」（Jayatilleke 1963 持肯定說）
- 是否為佛教獨創之「人格化煩惱」（Bodhi 2005 持肯定說）
- 是否有更早期之形式（少數學者推測可能承自吠陀文獻之相應比喻）

### 3. 「有 / 無有」（bhava / vibhava）之中道

dhp282「有 / 無有」之中道表述為《法句》中最具理論深度之段落。學界爭議：

- *Dhp-a* 解「vibhava」為「欲無」（kāma-vibhava）— 偏重倫理層面
- Jayatilleke（1963）解為形上學之「存在 / 非存在」對立 — 偏重形上學層面
- 現代多數學者（如 Bodhi 2005; Cousins 1996）採折衷解：「vibhava」兼具倫理（欲無）與形上（斷見）兩層

### 4. 「三法印」之概念史

dhp277-279「三法印」之標準化表述是否為佛陀原始教說？學界（Druart 2017; Bronkhorst 1993）爭議：

- 三法印之概念內涵於原始《尼柯耶》已具雛形（cf. SN 22.59 等）
- 但「三法印」之定型化（tilakkhaṇa）與「法印」（dhamma-uddesa）之名稱屬部派佛教
- 《法句》此處之「三頌結構」屬中間形式

### 5. 「隨眠」之分類（dhp339）

學界（Gethin 2001）指出《法句》此處使用「隨眠」（anusaya）一詞，但未列舉具體之「七隨眠」（貪瞋痴慢見疑戒禁取）。此反映《法句》之編纂年代早於「七隨眠」分類之定型（約阿毘達磨時期）。

### 6. 「三十六流」（dhp340）之具體所指

*Dhp-a* 解為「三十六種欲流」。學界爭議其可能為：（a）象徵性數字（cf. 《增支部》中其他象徵數字）；（b）佛教對印度古傳「三十六」之沿用；（c）對「六十二見」之另一分類。Jayatilleke（1963）認為後者可能性較高。

- **印度教《奧義書》之「欲」（kāma）**：古印度《奧義書》（cf. *Bṛhadāraṇyaka Upaniṣad* 1.2）亦以「欲」為束縛之源，但處理方式不同：奧義書以「知梵」（brahma-vidyā）為解脫，佛教以「八正道」為解脫。
- **耆那教之「漏」（āsava）**：耆那教亦使用「āsava」一詞，佛教與耆那教共承此印度古傳術語，但具體所指略異。學界（Dundas 2002, *The Jains*）指出耆那教之「āsava」含「四漏」（manah-paryaya-āsrava 等），佛教「四漏」（欲漏、有漏、見漏、無明漏）為不同分類。
- **婆羅門教之「梵行」（brahmacariya）**：本章 21 之「真比丘」（dhp267「修梵行者，稱為比丘」）所論之「梵行」非婆羅門教之「學生期禁欲」意，而是佛教之「出世梵行」（出離欲之修）。cf. MN 26 之「出家梵行」。

## 比丘品（dhp 360–382）段落註釋

### dhp 360–365：六根門制

「眼制為善」至「意制為善」並列六根——眼、耳、鼻、舌、身、意——皆「制」。巴利 saṃvara（制、律儀）不是感官禁絕（indriya-saṃvara ≠ indriya-samuccheda），而是對根境接觸時保持覺知，不讓貪憂趁機而入。巴利註書 *Dhammapada-aṭṭhakathā* 將此對應《中部》MN 152《根修習經》的教學架構。

「處處制之比丘，解脫一切苦」——「解脫一切苦」（sabbadukkha-ppamocana）涵蓋三苦：苦苦、壞苦、行苦。Sāratthadīpanī（5th c.）註：此處「制」與解脫是因果關係，非二事。

### dhp 366–368：法喜、念法

「法喜法樂」——dhammābhirata / dhammarata。巴利 dhamma 雙關：作為教法，亦作為所觀察的現象（dhamma = 法 = 對象）。此處連結《清淨道論》XXI 章的法隨念（dhammanusati）。

「正法不衰」——「正法」（saddhamma）= 佛陀教法；「不衰」（nappamajjati）= 不敗、不退。此句「正法」非指外在經典存佚，而是指法的實踐不退。

### dhp 369–371：不輕己、不嫉妒、三昧

「不得輕己所得，不行嫉妒於他」——《法句經》少數直接處理僧團共處倫理（saṃvāsa-ācāra）的段落。「嫉妒」（issā / macchariya）對應六十二見中的「嫉妒相應煩惱」。巴利註書將此連接僧團破裂（saṅgha-bheda）的預防。

「嫉妒他之比丘，不得三昧」——三昧（samādhi）與嫉妒（issā）的對立呼應《增支部》AN 5.73《三昧障礙經》：嫉妒是入定的五大障礙之一。

### dhp 372–374：名色、無我所、住慈

「於一切名色，無我所執」——名色（nāma-rūpa）= 五蘊中的四無色蘊（受想行識）+ 一色蘊（rūpa）的合稱；精確地說，此處特指「十二緣起」中的名色支。「無我所執」（nirāmisa）= 斷除對自我及其延伸的執著。

「住慈之比丘」——慈（mettā）在此特指四梵住（brahmavihāra）之一的慈無量。巴利 *Buddha-vagga* 早期註書將「住慈」解為五遍處（pañcakasiṇa）之一的慈遍處成就。

### dhp 375–377：舟喻、五五修

「當汲此舟」——「舟」（nāva）喻五蘊；對應《中部》MN 109《滿月大經》的「小舟喻」：比丘當勤汲舟中之水（漏，āsava），使舟輕可度瀑流。「貪與瞋」（lobha + dosa）對應兩種根本煩惱，加上癡（moha）= 三毒（akusala-mūla）。

「當斷五捨五，更修於五」——巴利學者主流解依 *Sāratthadīpanī* 與《清淨道論》XXII：
- **斷五** = 斷五下分結（orambhāgiya）：身見、疑、戒禁取、欲貪、瞋
- **捨五** = 捨五上分結（uddhambhāgiya）：色貪、無色貪、慢、掉舉、無明
- **修五** = 修五根（indriya）：信、進、念、定、慧

另一派解為「斷五蓋 / 捨五欲 / 修五根」，主流採前者。Bhikkhu Bodhi 在其 2000 年註譯本中採取「結解」派。

### dhp 378–382：獨修、不放逸、自庇護

「勿放逸吞鐵丸」——鐵丸喻（ayyūkoḷa）見於《相應部》SN 36.4《鐵丸喻經》：比丘若對欲塵起貪，如吞燒紅鐵丸而嚎哭。此喻針對感官欲望（kāmesu）的吸引力。

「入於空屋，心寂比丘」——空屋（suññāgāra）雙關：外在的修行處所，亦內心的空寂。對應《增支部》AN 5.176《空屋經》與《中部》MN 122《空經》（Suñña-sutta）的雙重意涵。

「自為己庇護」——自依止（attasaraṇa）原則，呼應《律藏》Māgandiyasutta 與《雜阿含》「自依止，法依止」（attadīpā, dhammadīpā）。此處是《法句經》中最接近「個體救度」立場的表述，學界對此是否與後世「自力」詮釋吻合有爭議。

## 婆羅門品（dhp 383–423）段落註釋

婆羅門品是《法句經》最長的一品，共 41 頌（依用戶結語），亦哲學密度最高。它以「我稱此為婆羅門」（taṁ ahaṁ brūmi brāhmaṇaṁ）為反覆出現的句式，重新定義「婆羅門」——從世襲的吠陀祭司階級，轉為「自證圓滿者」。

此一品集中展現佛教對婆羅門教的「內化轉向」（internalization turn）。學者認為這是早期佛教與《奧義書》共同的精神運動，但佛教拒絕《奧義書》的「我」（ātman）本體論基礎，將解脫建立在「無我」（anattā）上。

### dhp 383–384：忍耐、不瞋

「忍耐是力量」——khanti-bala。巴利傳統分三種忍：對瞋的忍、對苦的忍、對法的忍（dhammanijjhāna-khama）。此處特指對瞋的忍。「婆羅門」（brāhmaṇa）字面 = 神聖的知識擁有者，此處被翻轉為「自證者」。

「不瞋調柔」——akkodhana = 心柔軟不僵硬。Bhikkhu Bodhi 註此處呼應《長部》DN 34《十上經》（Dasuttara-sutta）的「不忍」項。

### dhp 385：蓮花喻

「如水蓮不染」——pokkharapatta = 蓮花。「蓮不染水」喻在印度文學中常見，但佛教版本強調「水不染蓮」——這是佛教的特殊版本。Sujato 註此喻針對感官欲望（kāmesu alimpana），而非精神逃避。與《摩訶婆羅多》「蓮出水」的印度教版本形成對照。

### dhp 386：知苦

「知此身之苦」——dukkha 在此具三層義：苦苦（dukkha-dukkha）、壞苦（vipariṇāma-dukkha）、行苦（saṅkhāra-dukkha）。「身」（kāya）= 五蘊之身。「於此」（idha）= 現生，非死後。「無重擔」（pannabhāra）喻斷輪迴之負擔。

### dhp 387：慧深

「慧深、巧於道非道」——gambhīra-pañña = 深慧；medhāvī = 慧解。「道非道」（maggāmagga）的識別是《清淨道論》核心主題：未至道而誤以為至道，是修行最大危險。

### dhp 388–389：居者、不居者

「不與居者及不住者混」——gahaṭṭha = 在家眾；anāgāra = 出家眾。「二者」（ubhaya）= 兩者皆離。Abhidhammattha-saṅgaha 將此視為「離二邊」（majjhimā paṭipadā）的中道觀：「不與世俗混，亦不與僧團糾紛混」。

「放下棍棒對眾生」——daṇḍa = 任何形式的暴力（身口意）。此頌是佛教「非暴力」（ahiṃsā）的核心表述，對應《正法念處經》「一切眾生皆畏殺」的立場。

### dhp 390–391：對敵無瞋、於欲不染

「於怨者無怨恨」——attadaṇḍa = 內在瞋恨武器。「怨者」（virodha）= 敵對者。佛法不僅禁外在的暴力，亦禁內在的瞋念。「sādāna / anādāna」= 取 / 不取。

「如蓮不染」——同 dhp 385 複頌，但對象更廣（kāmesu = 諸欲）。Buddha-vagga 對比組表明：本品以不同情境反覆「蓮不染」喻，建立解脫的視覺統一性。

### dhp 392–393：度欲、度貪

「度欲（kāma-bhava）→ 度貪（taṇhā-bhava）」——bhava 有兩層：存在（becoming）與輪迴（existence）。Sāratthadīpanī 將「kāmabhava」「taṇhābhava」分別解為欲界存在與貪驅動的存在。兩頌結構相同（「捨……出家遊行」），是佛教出離（nekkhamma）的標準表述。

### dhp 394：捨人間天上結

「斷人間結，越天上結」——mānusaka yoga = 人類特有的五欲與煩惱；dibba yoga = 天界特有的細微煩惱。此頌反映佛教對天界的非絕對化：天界不是終點，仍是輪迴。這與婆羅門教將天界視為較高但仍輪迴的立場接近，與耆那教類似，但佛教添加「非我」基礎。

### dhp 395：度瀑流

「度瀑流」——ogha = 瀑流。jhāyī = 禪思者，在此專指已得八定者。「無執」（anupādāya）= 不取著於五蘊；對應《相應部》SN 22.81《取著經》（Upādāna-sutta）。

### dhp 396–397：捨喜、捨恚

「捨喜捨恚」——rati = 對可喜事物的喜愛；arati = 對不可喜事物的厭惡。二者皆是煩惱的兩端。「清涼」（sītibhūta）是涅槃的另一稱謂：從煩惱之火回歸寂滅之涼。「勇猛勝一切世間」（sabbalokābhibhū vīra）——此句在原典中常被獨立引用，象徵阿羅漢的超越性。

### dhp 398–399：知眾生生死、漏盡

「知眾生之死」（cutiṁ vedi sattānaṁ）= 他心通 / 死生智；「知投生」（upapattiñca sabbaso）= 宿住隨念智 / 死生智的另一面。此二者加上天眼通（divyacakkhu）= 三明（tevijjā）。漏盡（khīṇāsava）= 阿羅漢位。「āsava」字根是 √sru（流動），喻煩惱如液體從傷口流出。

### dhp 400–402：知眾生之死、漏盡、純淨

「過去未來現在無一物」——akiñcana = 無一物屬於我；對應《相應部》SN 22.36《摩訶羅經》（Mahālī-sutta）。學者認為此頌呼應《奧義書》neti-neti（非如此非如此）的「遮詮」進路，但否定其「我」本體。

「如月清淨無染」——月喻見於多部經：《長部》DN 2《沙門果經》（Sāmaññaphala-sutta）「月輪」喻；《自說經》Udāna 11「月喻」。清淨（suddha / vippasanna）= 煩惱的徹底止息。

### dhp 403–404：度此岸、捨欲

「度此岸」——palipatha 字面意「壞路」、「邪道」；saṃsāra = 流轉、輪迴。

### dhp 405–407：度的對象

度的對象從欲（kāma）→ 貪（taṇhā）→ 人間結（mānusaka yoga）→ 天上結（dibba yoga）——層層出離。

### dhp 408–409：勇猛勝一切世間、勝世間

「勇猛勝一切世間」——loka = 三界：欲界、色界、無色界。「勝」（abhibhū）= 超越。「勇猛」（vīra）= 大丈夫。

### dhp 410–411：知眾生生死、漏盡

「知眾生之死」（cutiṁ vedi sattānaṁ）= 他心通 / 死生智。

### dhp 411–413：漏盡、純淨

「漏盡」（khīṇāsava）= 阿羅漢位。詳細含義見上 dhp 399 註。「無所有」（akiñcana）= 無一物屬於我。

### dhp 414：知前際、見天界地獄

「知前際、見天界與地獄、證生滅盡」——這是「三明」（tevijjā）：宿命明（pubbenivāsānussati）、天眼明（dibbacakkhu）、漏盡明（āsavakkhaya）。

### dhp 415–423：婆羅門品結尾

415「勇猛勝一切世間」（sabbalokābhibhū vīra）——總頌。

416–420 集中描述阿羅漢的智慧：知生死（cūti-upapatti）、漏盡、天眼、宿命通、無礙解脫。「usabha」= 公牛，象徵強而有力。「mahesi」= 大仙人。「nhātaka」= 已沐浴（出離塵垢之喻）。

421「我稱此為婆羅門」——「brūmi」（我說）= 權威性的宣布。「brāhmaṇa」在此完全脫離種姓意涵，成為純粹的精神位階。

### 結語註釋（dhp 末尾概要）

《法句經》末尾的「概要」（uddāna）列出全經 26 品、423 頌的結構。頌數統計略有出入：個別傳統計為 423 頌，另計為 425 頌（含歸敬偈）。學界共識：《法句經》以「雙品」（Yamakavagga）為起始，「婆羅門品」（Brahmavagga）為收束，呼應經名 Dhammapada——「法」與「句」的雙重結構。

結語中對每品頌數的統計（如「比丘品二十三頌」、「婆羅門品四十頌」）反映了部派佛教時代的編纂狀態。Lamotte 1944 指出，該統計與漢譯《法句譬喻經》（鳩摩羅什譯）的品數有差異，反映了上座部（Theravāda）與說一切有部（Sarvāstivāda）傳承的不同。

## 與其他經典的關聯（本段新增）

### 與巴利三藏其他經

- **《長部》DN 2《沙門果經》（Sāmaññaphala-sutta）**：佛陀列舉出家功德——本品的「禪」「定」「解脫」描述與該經一致
- **《中部》MN 27《象跡喻經》（Cūḷahatthipadopama-sutta）**：「如所有性」（yathābhūta）之智慧——本品的「正觀」「度瀑流」承襲此一進路
- **《增支部》AN 4.5《不眨眼經》（Ambalatthikārāhulavāda）**：對阿難說「應以不放逸律儀根門」——本品「六根門制」呼應
- **《相應部》SN 22.85《摩訶羅經》（Mahālī-sutta）**：對蘊的觀察——本品「度名色」承襲
- **《長部》DN 1《梵網經》（Brahmajāla-sutta）**：同樣以「婆羅門」為題——兩者同源於早期佛教對婆羅門教義的反思，但獨立發展

### 與漢譯佛典對照

- **《法句譬喻經》（Taisho 211）**：鳩摩羅什譯，廣本，多譬喻，與巴利本互為對照。結構差異反映上座部 vs 說一切有部對「法」的詮釋重點不同
- **《出曜經》（Taisho 212）**：竺佛念譯，與《法句經》對應，但頌數結構略異
- **《本事經》（Taisho 765）**：與《法句經》本生部分對應
- **《法集要頌經》（Taisho 213）**：施護譯，相當於巴利 Udāna 的一部分

Lamotte 1944、Saddhātissa 1985 對比研究指出，巴利本與鳩摩羅什譯本並非直譯關係，而是同一傳統的不同傳承。

### 與其他宗教對照

- **婆羅門教 / 印度教**：本品與《梨俱吠陀》《奧義書》《摩訶婆羅多》的「婆羅門」概念對話——印度教婆羅門以「生」（jaṭi / varṇa）為基礎，佛教以「證」（bhāvanā-maya）為基礎
- **耆那教**：同樣強調「自制」（samyama）與「不害」（ahimsā），但保留「我」（jīva）本體，與佛教「無我」相對
- **基督教**：「度彼岸」之喻與《新約》「過紅海」「出埃及」之喻可比，但二者的「彼岸」內涵不同：佛教是出輪迴，基督教是入天國

## 主要學術爭議（本段新增）

1. **「婆羅門」重定義的政治意涵**
   - D.D. Kosambi（1965）：此品反映佛教對婆羅門階級的反抗
   - Romila Thapar（1978）：佛教並非階級鬥爭，而是「重組文化符號」
   - Gananath Obeyesekere（1991）：佛教實際吸納了婆羅門文化

2. **「婆羅門品」與〈梵網經〉之關係**
   - 平川彰（Hirakawa 1990）：〈梵網經〉的「62 見」可能比本品更早定型
   - 學界主流：兩者同源於早期佛教對婆羅門教義的反思，但獨立發展

3. **「三明」是否限於佛與阿羅漢**
   - 上座部傳統：三明為佛所獨有，阿羅漢得「六通」而非「三明」
   - 大眾部 / 分別說部：阿羅漢亦得三明
   - 本品描述阿羅漢具三明，與大眾部傳統更接近

4. **「涅槃」的詮釋**
   - 學界爭議：涅槃是「存在」（ontological）還是「境界」（experiential）？
   - 上座部主流：涅槃是「煩惱的止息」，不涉本體論
   - 中觀 / 唯識傳統：涅槃是「實相」的顯現，涉本體論
   - 本品「貪瞋癡永盡」的描述屬前者

5. **「自為己庇護」是否為「自力」預表**
   - 上座部傳統：呼應「自依止，法依止」（attadīpā, dhammadīpā）
   - 大乘詮釋：常被引用為「自力」原則的預表，但巴利原典未見「他力」（paradīpa）對比，故此詮釋可疑
