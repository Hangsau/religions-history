### 第一品（KaU_1.1–1.29）

KaU_1.1 //
Uśan（烏善）後裔之 Vājaśravas（婆遮斯拉瓦斯），誠然，將一切所有布施出去；其人有一子，名 Naciketas（那吉迦多）。

KaU_1.2 //
那吉迦多尚年少，正當（祭禮之）布施被引領而出之時，śraddhā（信心）入於其心。

KaU_1.3 //
飲過水的、嚼盡枯草的、奶已擠盡的、感官俱失的；「無樂」之名為彼等世界；施予此等（牛）者，將往彼處。

KaU_1.4 //
他對父親說：「父親，您將我送給誰呢？」第二遍、第三遍。他對父親說：「我將你送給 Yama（閻摩，死神）。」

KaU_1.5 //
「我赴諸多之中而為第一，我赴諸多之中而居中；究竟有何當為閻摩所作者，今日我便能做。」

KaU_1.6 //
瞻前顧後，如先人、如後人之所見；凡人如穀熟，穀熟亦再生。

KaU_1.7 //
Vaiśvānara（遍照者之火）以客人之身進入婆羅門之家；人為其作此安撫：「取 Vaivasvata（閻摩之裔）之水來。」

KaU_1.8 //
於期盼中，集聚真實之言、祭祀之福、妻兒牛畜一切；小智者取此（組合）為用，無食而居婆羅門於家中。

KaU_1.9 //
在我家三夜無食之婆羅門客人，禮敬可敬；向你致敬啊婆羅門！願我平安！因此請你賜我三願。

KaU_1.10 //
心意澄明、安詳決意、無有忿怒的 Gautama（喬答摩）啊，願勿使我為您遣送的閻摩所觸；我所領受者乃您所遣之事；三願之首，我選擇此第一願。

KaU_1.11 //
如前所述，Uddālaki 之子 Āruṇi 為您所遣；無忿安眠諸夜，您已親見我從閻摩之口釋出。

KaU_1.12 //
於天界無有少分恐懼；你在彼處不老不畏；度越飢渴二者，超越愁苦，於天界歡喜。

KaU_1.13 //
閻摩啊！你已教我以天界之火，我信於此，請告知我。享天界、得永生者——我以此第二願選擇。

KaU_1.14 //
我對你說，請你諦聽；那吉迦多！知天界之火；無盡世界的獲得、究竟的安住——你要知道此隱於洞窟中。

KaU_1.15 //
他便告訴他：磚塊若干，大小若干，如何佈置。他亦如所言答覆之；閻摩復欣然啟口。

KaU_1.16 //
大我者（mahātmā）喜悅地對他說：「今日我再賜你一願；此火將以你名（那吉迦多）而立；並接受此具多形之祭祀（sṛṅkā）吧！」

KaU_1.17 //
具三那吉迦多祭者（triṇāciketas），以三（時）入三祭合，行三業（trikarmakṛt），度過生死；知 Brahman（梵）為祭祀之主的、可敬之神（deva-īḍya），得此究竟安息。

KaU_1.18 //
具三那吉迦多祭者，知此三者，即使知者亦（三日）焚那吉迦多祭；推開閻摩之索於前，超愁苦，於天界歡喜。

KaU_1.19 //
此即你那吉迦多之天界火，你以第二願所選；此火，人將說「是你的」；那吉迦多！請選第三願。

KaU_1.20 //
死後之人是否有（自我）？此處有疑；有人說「有」，有人說「無」；你所教之智，我欲知；此乃願望之第三。

KaU_1.21 //
諸神昔於此亦曾猶豫；此法微妙，誠不易知；那吉迦多！請選另一願，勿使我被拋棄，勿放棄我。

KaU_1.22 //
諸神昔於此亦曾猶豫，閻摩啊！你說此不可知；而能說此者如你者不可得；無他願等於此。

KaU_1.23 //
選有百歲壽命之子孫吧！許多牛馬、象、黃金；廣大久居之地，請選擇！如你所欲，長久活命。

KaU_1.24 //
你若以此為同等之願，請選財富與長壽；於大地上，那吉迦多！你可為主人；我使你成為欲求之享受者。

KaU_1.25 //
你可隨心所欲地祈求一切難得之欲；這些配車配琴的女子，伴以鼓聲，凡人不可得；以我所賜，享受吧！那吉迦多！你未曾求不死。

KaU_1.26 //
凡人壽命有限，感官終將衰老；你之全壽命亦短促，那吉迦多！車馬歌聲為你所有。

KaU_1.27 //
人不可因財富而滿足；我們已得財富，我們已見你；你尚活一日，我們便活一日；唯有願望，方為我選。

KaU_1.28 //
凡人衰老短命，歸於不死；明知形色欲樂之事於長生之中，誰能歡喜？

KaU_1.29 //
在那大死轉移之處，閻摩啊！告我此事；我所選之願，隱而難入；那吉迦多不選其他。

// iti prathamā vallī //

### 第二品（KaU_2.1–2.25）

KaU_2.1 //
「餘為善（śreyas），餘為悅（preyas）；二者於人現異義；取善者善成，取悅者失義。」

KaU_2.2 //
善與悅皆臨於人；智者審度而分辨；智者取善，愚者取悅以為守護（yoga-kṣema）。

KaU_2.3 //
你啊那吉迦多！思慕可愛之相與可愛之欲，已超越之；此財之誘餌，眾多世人陷溺其中，你未得。

KaU_2.4 //
兩端遠離，異途而行；無明（avidyā）與明（vidyā）二者皆知；那吉迦多！求明者，我以為非貪愛欲者。

KaU_2.5 //
居於無明之中，自以為聰明，以為智者；愚蒙人如盲人牽盲人，徘徊受難。

KaU_2.6 //
愚者溺於財迷，不見來世；此世無、彼世無，以此自矜；復復陷於我之羈絆。

KaU_2.7 //
雖多聞不為多數所得，聽者眾多而不識；說者希奇，善說者得之；識者希奇，受善教者得之。

KaU_2.8 //
此非凡下之人所宣說；多番思惟，難以深知；非由他人所宣，無由得入；甚小不可思量，超過度量。

KaU_2.9 //
此理非以思辨得；唯由他人所宣，乃能了知。那吉迦多！你已得真持（satyadhṛti），願如你者更多，我所愛之人！

KaU_2.10 //
我知，此世所得皆無常（anitya）；非常（adhruva）不得常（dhruva）。故我以那吉迦多火，以無常之柴薪，已得常。

KaU_2.11 //
欲之成就、世界之安住、智之無盡、無畏之彼岸——見此偉大歌讚之根本，我以堅毅，那吉迦多！超越之。

KaU_2.12 //
那難見的、隱而入的、居於洞窟的、深藏於穴中的、古老者——以內在瑜伽證得此神（deva），智者便捨喜與悲。

KaU_2.13 //
凡人聞此，深記於心，奉行此微細之正法（dharma）；得此可喜之物，歡喜；我以為那吉迦多乃居於開顯之宮。

KaU_2.14 //
「此異於正法（dharma），異於非法（adharma），異於所作與未作（kṛtākṛta），異於已存在及將存在者；你所見者，請說之。」

KaU_2.15 //
「一切吠陀所稱之句，一切苦行所言；願行梵行者之所願；我簡要告訴你此句——『Om』即此。」

KaU_2.16 //
「此即是不滅（akṣara）之 Brahman，此即是不滅之至上；知此不滅者，凡其所求，皆得之。」

KaU_2.17 //
「此乃最勝依止，此乃至上依止；知此依止者，於梵界受尊榮。」

KaU_2.18 //
「智者（vipaścin）不生亦不死，從未曾生於任何處；無生、永恆、常住、古老；身被殺時，彼不被殺。」

KaU_2.19 //
「若殺者以為殺，被殺者以為被殺；二者皆不知；彼不殺亦不被殺。」

KaU_2.20 //
「比原子（aṇu）更微，比宏大更宏；ātman 居於有情之窟穴；無為者見之，離愁者，由元素（dhātu）之清淨，得自體之偉大。」

KaU_2.21 //
「坐者遠行，臥者遍往；那位神，除我之外，誰能使其醉？」

KaU_2.22 //
「無體而居於諸體中，不安而安住；智者知此 ātman 為宏大、遍在，便不憂悲。」

KaU_2.23 //
「此 ātman 非以教說得，非以智慧（medhā），非以多聞得；唯彼所選者，乃得之；彼（ātman）選此人，乃顯其自身。」

KaU_2.24 //
「不離惡行、不安、不集中、心不寧者，不能以智慧（prajñāna）得之。」

KaU_2.25 //
「對彼，梵與剎帝利二者皆成食糧；死為其佐料；如此，誰知其在何處？」

// iti dvitīyā vallī //

### 第三品（KaU_3.1–3.17）

KaU_3.1 //
「於善行世界，飲真理，進入洞窟中，於至上彼岸；博學梵者說：影與光、五火、三那吉迦多。」

KaU_3.2 //
「祭祀者之橋、不滅之梵、至上者；求無畏者之彼岸，我們能否得那吉迦多？」

KaU_3.3 //
「知 ātman 為乘者（rathin），身體為車（ratha）；智慧（buddhi）為御者（sārathi），意（manas）為韁繩（prahara）。」

KaU_3.4 //
「諸根（indriya）名為馬，所緣（viṣaya）為馳處；具 ātman、根、意相連者，智者稱為享者（bhoktṛ）。」

KaU_3.5 //
「無智者，常以不攝意；其諸根不制御，如御者之劣馬。」

KaU_3.6 //
「智者，常以攝意；其諸根可制御，如御者之良馬。」

KaU_3.7 //
「無智者，無意、恆不淨；不得此處（padam），入輪迴（saṃsāra）。」

KaU_3.8 //
「智者，有意、恆清淨；得此處，不再生（punar-janma）。」

KaU_3.9 //
「以智慧為御者，意攝韁繩之人；得道之彼岸，即 Viṣṇu（毘濕奴）至上之位（paramam padam）。」

KaU_3.10 //
「諸根之上為所緣，所緣之上為意，意之上為智慧，智慧之上為大 ātman（mahān ātman）。」

KaU_3.11 //
「大 ātman 之上為未顯者（avyakta），未顯者之上為人（puruṣa）；人之上無更勝；此為究竟（kāṣṭhā），此為至上歸趣（parā gati）。」

KaU_3.12 //
「此 ātman 隱於一切有情，不顯；以精微智慧，慧眼者見之。」

KaU_3.13 //
「智者以語意攝；以智攝於 ātman 中；以大者攝智；以寂靜攝於 ātman。」

KaU_3.14 //
「起！覺！得諸願，了知！剃刀之刃，銳利難越；智者言：此乃難行道。」

KaU_3.15 //
「無聲、無觸、無形、不滅、無味、恆常、無臭；無始無終、超越大者、恆常；知此者，從閻摩之口得解脫。」

KaU_3.16 //
「閻摩所言那吉迦多古說（nāciketam upākhyāna）；智者說已聽已，於梵界受尊榮。」

KaU_3.17 //
「於婆羅門會中，或於正信之時，宣此至上祕密；此人得無盡（ānantya）。」

// iti tṛtīyā vallī //

### 第四品（KaU_4.1–4.15）

KaU_4.1 //
「自生者（svayaṃbhū）將諸根（kha）外轉（vyatṛṇat）；故人見外而非內（antar-ātman）；少數智者（dhīra），轉眼向內，欲得不死（amṛtatva），見 ātman。」

KaU_4.2 //
「愚者（bāla）隨外馳之欲，往於閻摩張開之網；智者（dhīra）知不死（amṛtatva），於無常中不求常。」

KaU_4.3 //
「以彼（ātman）知形（rūpa）、色（rasa）、香（gandha）、聲（śabda）、觸（sparśa）、淫（maithuna）；此外更有何存？——此即是彼（etad vai tat）。」

KaU_4.4 //
「夢終、覺終，二者皆以彼見；知此 ātman 為宏大、遍在（vibhu）；智者不憂。」

KaU_4.5 //
「知此蜜杖之 ātman、生命者（jīva）之親近者、未來過去之主宰者（īśāna）；彼無所畏——此即是彼。」

KaU_4.6 //
「昔由苦行（tapas）而生者，先於水（adbhya）而生者；入洞窟而安住者，以諸有情而見者——此即是彼。」

KaU_4.7 //
「Aditi（阿底提）化為氣（prāṇa），具神性（devatāmayī）；入洞窟而安住者，以諸有情而生者——此即是彼。」

KaU_4.8 //
「Jātavedā 藏於二木間（araṇi）；如胎藏於孕婦中；日日可敬，覺者、具供者（haviṣmat）、人所祭火——此即是彼。」

KaU_4.9 //
「日從彼出，日入於彼；諸天皆安置（arpita）於彼；無有逾越（na atyeti kaścana）——此即是彼。」

KaU_4.10 //
「此處所有，彼處亦有（amutra）；彼處所有，此處亦隨之（anv iha）；死中得死（mṛtyuḥ sa mṛtyum āpnoti），於此視為異者。」

KaU_4.11 //
「唯以意（manas）得此；於此無異（nāna）可得；死中得死，於此視為異者。」

KaU_4.12 //
「拇指大小之人（aṅguṣṭhamātraḥ puruṣa），居於 ātman 之中；未來過去之主宰者；彼無所畏——此即是彼。」

KaU_4.13 //
「拇指大小之人，如無煙之火（jyotir iva adhūmaka）；未來過去之主宰者；彼即是今日，彼即是明日——此即是彼。」

KaU_4.14 //
「如雨降於險地（durge），分流於諸山；正法（dharma）亦然，見者（paśyams）分流各歸之。」

KaU_4.15 //
「如淨水注於清水（śuddhe śuddham āsiktam），味同此；智者知 ātman，喬答摩啊！亦復如是。」

// iti caturthī vallī //

### 第五品（KaU_5.1–5.15）

KaU_5.1 //
「十一門之城（pura ekādaśadvāra），無生者（aja）之曲意（avakracetas）；實踐之（anuṣṭhāya），無憂；解脫而得解脫——此即是彼。」

KaU_5.2 //
「天鵝（haṃsa）居於清淨（śuciṣat），居於空（vasur antarikṣasat），居於祭火（hotā vediṣat），居於客（atithir duroṇasat），居於人（nṛṣat），居於至上（varasat），居於真理（ṛtasat），居於太空（vyomasat）；生於水（abja），生於牛（goja），生於真理（ṛtaja），生於山（adrija），偉大真理。」

KaU_5.3 //
「上氣（prāṇa）上引（unnayati），下氣（apāna）下引（pratyag asyati）；居中之小者（vāman）坐著，諸天（viśve devā）皆敬奉之。」

KaU_5.4 //
「於此身內居者（dehin），將從身出離（vimucyamānasya）；此外更有何存？——此即是彼。」

KaU_5.5 //
「凡人非以氣、非以下氣而活；乃依彼（他者），二者皆依（upāśrita）。」

KaU_5.6 //
「我將說此祕密（guhyam）、古老之梵（brahma sanātanam）；如得死後，ātman 將如何，喬答摩啊！」

KaU_5.7 //
「有人入另一胎（yoni）以得身；有人依業、依學（yathākarma yathāśrutam）入柱（sthāṇu，堅住之物）。」

KaU_5.8 //
「彼於睡者中覺醒（supteṣu jāgarti），以欲造欲（kāmam kāmam puruṣo nirmimāṇaḥ）；彼即光明（śukra），彼即是 Brahman，彼稱為不死（amṛta）；一切世界依之而立，無有逾越——此即是彼。」

KaU_5.9 //
「如火入萬物，形對形而成相（rūpam rūpam pratirūpo babhūva）；如是，一切有情之內 ātman，形對形而成相，亦在內外。」

KaU_5.10 //
「如風（vāyu）入萬物，形對形而成相；如是，一切有情之內 ātman，形對形而成相，亦在內外。」

KaU_5.11 //
「如日（sūrya）為一切世界之眼，不沾眼外諸垢（cākṣuṣair bāhyadoṣaiḥ）；如是，一切有情之內 ātman，不沾世間苦惱（lokaduḥkhena）。」

KaU_5.12 //
「唯一自在者（eko vaśī），一切有情之內 ātman，作多形為一形（ekaṃ rūpaṃ bahudhā yaḥ karoti）；智者（dhīra）見其居於 ātman 者，得常樂（śāśvataṃ sukham）；其餘不然。」

KaU_5.13 //
「於無常中永恆（nityaḥ anityānām），於有識中具識（cetanaḥ cetanānām），唯一於多者，置諸欲（kāmān vidadhāti）；智者見其居於 ātman 者，得常寂（śāśvatī śāntiḥ）；其餘不然。」

KaU_5.14 //
「彼是『不可指示』（anirdeśya）之至上樂（paramaṃ sukham）；如何得知？如何顯？如何照？」

KaU_5.15 //
「彼處日不照，月與星不照；此閃電不照，何況此火；彼照，萬物隨之而照；一切由其光照而顯。」

// iti pañcamī vallī //

### 第六品（KaU_6.1–6.18）

KaU_6.1 //
「上根（ūrdhvamūla）下枝（avākśākha），此是永恆（sanātana）之 aśvattha（阿湿伐達樹）；彼即光明，彼即是 Brahman，彼稱為不死；一切世界依之而立，無有逾越——此即是彼。」

KaU_6.2 //
「此一切世界，從氣息（prāṇa）出而動搖（ejati niḥsṛtam）；偉大可畏（mahad bhayaṃ），金剛（vajra）高舉（udyatam）；知此者，得不死。」

KaU_6.3 //
「畏故（bhayāt）火燃，畏故日曬；畏故 Indra（因陀羅）與 Vāyu（風），乃至第五閻摩奔走（dhāvati）。」

KaU_6.4 //
「若此身解體前（prāk śarīrasya visrasaḥ）不能知；則於再生之世界（sargeṣu lokeṣu），又得身（śarīratvāya kalpate）。」

KaU_6.5 //
「如鏡中（adarśe），如夢中（svapne），如祖界（pitṛloke）；如水中影像（apsu parīva dadṛśe）；如影與光之於 Gandharva（乾闥婆）界；如是 Brahmaloka（梵界）。」

KaU_6.6 //
「知諸根個別之相（pṛthagbhāvam），個別生滅（udayāstamayau）；智者不憂。」

KaU_6.7 //
「諸根之上為意（manas），意之上為純質（sattva uttama）；純質之上為大 ātman；大 ātman 之上為未顯者（avyakta uttama）。」

KaU_6.8 //
「未顯者之上為人（puruṣa），遍在（vyāpaka），無相（aliṅga）；知此者，得解脫（mucyate），得不死（amṛtatvaṃ ca gacchati）。」

KaU_6.9 //
「彼無形可立（na saṃdṛśe tiṣṭhati rūpam asya），無眼可見（na cakṣuṣā paśyati kaścanainam）；以心（hṛdā）、慧（manīṣā）、意（manas）攝之（abhikḷpta）；知此者，得不死。」

KaU_6.10 //
「當五（智）連意皆立（pañcāvatiṣṭhante jñānāni manasā saha），慧亦不動（buddhiś ca na viceṣṭati）；此乃至上歸趣（parāṃ gatim）。」

KaU_6.11 //
「此為人所稱之 yoga（瑜伽）——穩固根之攝持（indriyadhāraṇām）；彼時瑜伽不分心（apramattaḥ）；蓋瑜伽有起有盡（prabhavāpyayau）。」

KaU_6.12 //
「彼非以語、非以意、非以眼可得；說『有』（asti iti）於他處（anyatra），如何得之？」

KaU_6.13 //
「以『有』得（upalabdhavyaḥ），以『有』之實相（tattvabhāvena）；於『有』得者，實相得確立（prasīdati）。」

KaU_6.14 //
「當心所住之一切欲（kāmā ye asya hṛdi śritāḥ）皆解脫；凡人（martyaḥ）便成不死（amṛta）；於此得梵（brahma samaśnute）。」

KaU_6.15 //
「當心中一切結（granthayaḥ）皆斷（prabhidyante）；凡人便成不死；此乃教誨之究竟（anuśāsanam）。」

KaU_6.16 //
「心有百零一脈（śataṃ caika ca hṛdayasya nāḍyaḥ）；其一出於頭頂（mūrdhānam abhiniḥsṛtaikā）；上趣此者（tayordhvam āyan）得不死；其餘則於離出（viṣvaṅk anyā utkramaṇe bhavati）。」

KaU_6.17 //
「拇指大小之人，內 ātman，常居眾人心（hṛdaye saṃniviṣṭaḥ）；如從草鞘（muñjāt）抽草（iṣīkām）以出於身，以堅毅（dhairyeṇa）為之；知彼為光明、不死（śukram amṛtam）；知彼為光明、不死。」

KaU_6.18 //
「那吉迦多得閻摩所教之智、yoga 之全法；得梵、無染（vira-ja）、不死（vimṛtyu）；其他知此內在者亦然。」

// iti ṣaṣṭhī vallī //
