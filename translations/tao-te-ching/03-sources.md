# 道德經 — 原文版本說明

> 版本選擇、抄本關係、來源

## 1. 採用版本

**主版本：魏·王弼本**（~3 世紀）

理由：
- 自晉唐以來通行版本
- 注疏傳統最完整
- 後世幾乎所有英譯、歐美研究以此為底

## 2. 重要版本對照

| 版本 | 時代 | 性質 | 對王弼本差異 |
|------|------|------|------------|
| **郭店楚簡本** | ~300 BC（戰國）| 現存最早抄本（1993 荊門出土）| 文字簡古，章節數少（~2000 字）|
| **馬王堆帛書甲乙本** | ~200 BC（西漢）| 1973 長沙出土 | 乙本「德經」在前、「道經」在後；若干字異 |
| **河上公本** | ~1 世紀（傳統）| 早期注本 | 文字與王弼本略異；偏養生家立場 |
| **嚴遵本** | 1 世紀 BC | 最早注本 | 章節次序與王弼本略異 |
| **想爾本** | ~2 世紀 | 道教化改寫 | 「道」常改「一」 |
| **王弼本** | ~3 世紀 | 玄學注本 | 後世通行祖本 |
| **唐玄宗御注本** | 735 | 官方注 | 唐代官定本 |
| **敦煌寫本** | ~5-10 世紀 | 多種殘卷 | 顯示早期抄本分歧 |

## 3. 文本歧義類型

### 3.1 通假字 / 古今字

- 「道可道，非常道」中「常」=「恆」（漢避文帝諱改）
- 「萬物作焉而不辭」中「辭」=「始」或「治」（郭店本作「始」）
- 「躁勝寒」中「躁」=「燥」（郭店本）

### 3.2 章節次序

- 帛書本「德經」在前、「道經」在後
- 第 39 章王弼本與帛書本斷句略異
- 第 67 章王弼本「天下皆謂我大」— 帛書本「我」作「我道」

### 3.3 虛詞歧義

- 「之 / 乎 / 者 / 也」— 早期文本虛詞用法與後世不同
- 例：「多言數窮，不如守中」中「中」=「沖」（虛）— 帛書本作「沖」

## 4. 參考英譯（僅作交叉驗證）

- **James Legge**（1891）：第一本系統英譯；偏字面直譯；19 世紀傳教士視角
- **Arthur Waley**（1934）：文學化意譯；影響 T.S. Eliot 詩作
- **Wing-tsit Chan**（1963）：哲學化翻譯；學術標準
- **D.C. Lau**（1963）：學術標準；對照王弼本與帛書本
- **Robert Henricks**（1989）：基於馬王堆帛書甲本
- **Edward Slingerland**（2003）：哲學注釋版，選王弼本
- **Addiss & Lombardo**（1993）：禪意譯本
