## 1. 目標

建立一份**蒐集性質**的世界宗教與經文總目錄：

- **宗教**：列舉世界所有主要 + 次要宗教（含分支、教派）
- **經文**：列舉各宗教核心 + 次要經典（含年代、語言、版本、現存狀態）
- **最後階段**：心理學內容（宗教心理學 / 靈性心理學 / 信仰與行為）

**性質**：清單 + 簡介 + 來源 — 不是深度論文（深度研究後續 P 階段補）。

## 3. 工作方法

### 3.0 翻譯原則（2026-06-29 修訂）

**整個計畫不使用已翻譯的譯文。原文優先 → AI 翻譯 → 交叉比對。**

- **第一步：原文** — 抓原語言文本（梵文、阿拉伯、希伯來、古典漢語、巴利、希臘等）
- **第二步：AI 翻譯** — 用 LLM 直接從原文翻成繁中白話
- **第三步：交叉比對** — 拿現有譯本（Legge、KJV、Griffith 等）只作 cross-check，不主用
- **第四步：記錄差異** — 翻譯中遇到的歧義 / 多解 / 學界爭論 → 寫進註腳

**為什麼**：
- 現成譯本帶翻譯者神學 / 文化立場（如 KJV 對教派、LXX vs MT 舊約差異）
- AI 翻譯獨立於教派，可作「中立第三方」對照
- 交叉比對才能標出爭議

詳細 SOP 見 [`methodology/translation-workflow.md`](./methodology/translation-workflow.md)。

### 3.1 資料來源（嚴格分流）

- **A 級**：原文資料庫（VIAF、GRETIL、SEFARIA、Perseus、SAT、CBETA、ctext.org 等）
- **B 級**：學術專著、Stanford Encyclopedia of Philosophy、Encyclopaedia Britannica
- **C 級**：大學公開課、博物館展覽
- **D 級**：中文百科 — 僅 cross-check
- **E 級**：教內自述 — 標明「教內視角」
- **X 級**：現成譯本 — **只用於交叉比對**，不直接引用

### 3.2 清單格式（宗教條目）

```markdown
### [宗教名稱]
- **起源**：年代 / 地點
- **創立者 / 神話源**：
- **核心信條**（3-5 點）：
- **核心經典**：
- **主要分支**：
- **當代信徒規模**（最新可查）：
- **狀態**：活躍 / 衰退 / 已消亡 / 融合
- **來源等級**：A / B / C / D
```

### 3.3 清單格式（經文條目）

```markdown
### [經典名稱]
- **所屬宗教**：
- **成書年代**：
- **原始語言**：
- **核心內容**（一句話）：
- **結構**：章節 / 卷數
- **現存狀態**：完整 / 部分 / 散佚後重建
- **權威性**：教內 / 學界
- **來源等級**：
```

### 3.4 爭議處理原則

- 史料衝突 → 兩說並列，註明出處
- 教內 vs 學界 觀點不同 → 標明視角
- 在世宗教 / 活人 → 避免價值判斷，僅描述事實

## 7. Anti-pattern

- ❌ 引用單一立場來源
- ❌ 簡轉繁混用
- ❌ 一篇塞整個宗教（>3000 行）— 拆 sub-page
- ❌ 用 emoji 緩衝內容
- ❌ 自我標榜「中立」而不標明視角
- ❌ 「我不確定」當免責符（真不確定就標 "資料待補"）
