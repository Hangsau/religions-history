# 宗教史 + 經文研究計畫

> 從史前到現代：所有宗教 + 所有經文的結構化蒐集
> 啟動：2026-06-29
> 心理學內容最後階段加入（先預留位置）

---

## 1. 目標

建立一份**蒐集性質**的世界宗教與經文總目錄：

- **宗教**：列舉世界所有主要 + 次要宗教（含分支、教派）
- **經文**：列舉各宗教核心 + 次要經典（含年代、語言、版本、現存狀態）
- **最後階段**：心理學內容（宗教心理學 / 靈性心理學 / 信仰與行為）

**性質**：清單 + 簡介 + 來源 — 不是深度論文（深度研究後續 P 階段補）。

## 3. 工作方法

### 3.1 資料來源（嚴格分流）

- **A 級**：學術專著、Stanford Encyclopedia of Philosophy、Encyclopaedia Britannica
- **B 級**：大學公開課、博物館展覽
- **C 級**：中文百科 — 僅 cross-check
- **D 級**：教內自述 — 標明「教內視角」

### 3.2 清單格式（宗教條目）

```markdown
### [宗教名稱]
- **起源**：年代 / 地點
- **創立者 / 神話源**：
- **核心信條**（3-5 點）：
- **核心經典**：
- **主要分支**：
- **當代信徒規模**（最新可查）：
- **狀態**：活躍 / 衰退 / 已消亡 / 融合
- **來源等級**：A / B / C / D
```

### 3.3 清單格式（經文條目）

```markdown
### [經典名稱]
- **所屬宗教**：
- **成書年代**：
- **原始語言**：
- **核心內容**（一句話）：
- **結構**：章節 / 卷數
- **現存狀態**：完整 / 部分 / 散佚後重建
- **權威性**：教內 / 學界
- **來源等級**：
```

### 3.4 爭議處理原則

- 史料衝突 → 兩說並列，註明出處
- 教內 vs 學界 觀點不同 → 標明視角
- 在世宗教 / 活人 → 避免價值判斷，僅描述事實

## 4. 階段規劃

| 階段 | 範圍 | 預期產出 |
|------|------|---------|
| **P0** | 框架 + PLAN + 結構 | PLAN + CLAUDE + README + 00-overview | ✓ 完成 |
| **P1** | 宗教總目錄 | `00-overview/religions-inventory.md`（≥50 條目）|

| **P3** | 各區域細節 | 每個區域一個 overview.md + 主要宗教子條目 |

| **P5** | 交叉專題 | 宗教與政治、宗教與科學、末世論比較 |
| **P6** | 心理學（延後）| 11-psychology/ 目錄建立 + 主要分支介紹 |
| **P7** | 開放資料 / 視覺化 | 知識圖譜、引用檔、英文版 |

## 5. 維護機制

- **更新觸發**：新資料、新研究、爭議事件、user 反饋
- **版本控制**：git，每完成一個 P 階段 commit + push
- **品質自檢**：每批清單交付前跑 §6 自檢
- **INDEX.md 更新**：完成 P1/P2/P3 必更新 `~/.claude/memory/INDEX.md` 專案地圖

---

## 7. Anti-pattern

- ❌ 引用單一立場來源
- ❌ 簡轉繁混用
- ❌ 一篇塞整個宗教（>3000 行）— 拆 sub-page
- ❌ 用 emoji 緩衝內容
- ❌ 自我標榜「中立」而不標明視角
- ❌ 「我不確定」當免責符（真不確定就標 "資料待補"）
